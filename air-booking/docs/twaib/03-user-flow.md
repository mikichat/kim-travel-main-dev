# 03-user-flow.md — 사용자 흐름도

## 주요 시나리오

### 시나리오 1: 새 안내문 생성 (기본 흐름)

```
관리자
  ↓
[안내문 탭 열기]
  ↓
입력 영역
  ├─ 여행지 입력: "필리핀"
  ├─ 기간 입력: "2024-12-18" ~ "2024-12-22"
  ├─ [항공편 불러오기] 버튼 클릭
  │   ↓
  │   팝업: flight_schedules 목록
  │   ├─ (검색) "AAL" 필터
  │   ├─ 선택: "AAL123 / 2024-12-18 / ICN→MNL"
  │   └─ [선택] → 팝업 닫기
  │     ↓
  │     항공편: 자동 채움
  │     출발시간: 02:00 (자동)
  │     여행지: 필리핀 (자동 추론)
  │     기간: 2024-12-18 ~ 2024-12-22 (자동)
  ├─ 출발지 입력: "함열스포츠센터"
  ├─ 경비 입력: "2,380,000"
  ├─ 섹션 토글 확인 (기본: 모두 On)
  │   ☑ 일반 안내사항
  │   ☑ 날씨
  │   ☑ 복장 / 준비물
  │   ☑ 환율
  │   ☐ 수하물 정보
  │   ☐ 커스텀 공지
  └─ [AI 안내문 생성] 버튼 클릭 (진행바 표시)
    ↓
API: POST /api/travel-guides/generate
  {
    destination: "필리핀",
    start_date: "2024-12-18",
    end_date: "2024-12-22",
    departure_place: "함열스포츠센터",
    departure_time: "02:00",
    cost_per_person: 2380000
  }
  ↓ (3~5초)
Gemini API 응답: { weather, outfit, checklist, currency, general_notice }
  ↓
미리보기 표시
  ├─ 히어로 섹션 (배경: 필리핀 이미지, 자동)
  │   필리핀 5일 여행
  │   2024-12-18 ~ 2024-12-22
  ├─ 일반 안내사항
  │   "필리핀은 동남아시아의 따뜻한 섬나라입니다..."
  │   [수정]
  ├─ 날씨 (12월)
  │   "평균 기온: 28°C, 습도: 75%"
  │   [수정]
  ├─ 복장
  │   ✓ 반팔/민소매 셔츠
  │   ✓ 린넨 바지
  │   [수정]
  ├─ 준비물 필수
  │   ✓ 여권
  │   ✓ 비자
  │   [수정]
  ├─ 환율 (2024-12 기준)
  │   "1필리핀 페소 ≈ 24원"
  │   [수정]
  └─ 푸터 (여행사 로고)
    ↓
관리자 검토 (수정 필요 여부 판단)
  ├─ OK → [저장] / [이미지 복사]
  │   ↓
  │   ([저장] 선택 시)
  │   제목 입력: "필리핀 5일 여행" (기본값)
  │   [저장] → POST /api/travel-guides → 200 OK
  │   "저장되었습니다!"
  │   ↓
  │   ("이미지 복사" 선택 시)
  │   html2canvas로 미리보기 → Blob
  │   → navigator.clipboard.write() → 클립보드 복사
  │   "이미지가 클립보드에 복사되었습니다. 카톡에 붙여넣으세요."
  │   ↓
  │   카톡 열기 → 고객 채팅 선택 → Ctrl+V → 이미지 붙여넣기
  │   ↓
  │   고객 수신 (카톡 이미지)
  │
  └─ 수정 필요 → [섹션별 수정]
    ↓
    [날씨 섹션 [수정] 클릭]
      ↓
      모달: "날씨 편집"
      현재 내용: "평균 기온: 28°C, 습도: 75%"
      [수정]
      새 내용: "12월은 건기로 맑은 날씨가 대부분입니다. 평균 기온은 28°C 정도입니다."
      [저장] → aiContent.weather 업데이트
      ↓
      미리보기 실시간 반영
    ↓
    [저장] / [이미지 복사]
      ↓
      (흐름 동일)
```

### 시나리오 2: 이전 안내문 재사용

```
관리자
  ↓
[안내문 탭 열기]
  ↓
[저장된 안내문 목록] 버튼 클릭
  ↓
팝업: 저장된 안내문 목록
  GET /api/travel-guides?page=1&limit=10
  ↓
리스트 표시
  [필리핀 5일 여행] 2024-12-18 [편집] [삭제]
  [일본 3일 여행] 2024-11-20 [편집] [삭제]
  [홍콩 2일 여행] 2024-10-15 [편집] [삭제]
  ↓
[필리핀 5일 여행] 클릭
  ↓
입력 필드 자동 채움:
  여행지: 필리핀
  기간: 2024-12-18 ~ 2024-12-22
  항공편: AAL123 (자동)
  출발지: 함열스포츠센터
  경비: 2,380,000
  섹션 설정: 저장된 그대로
  ↓
미리보기: 저장된 ai_content + custom_content 표시
  ↓
필요시 [섹션별 수정] → 미리보기 반영
  ↓
[이미지 복사] → 클립보드 → 카톡 전송
```

### 시나리오 3: 섹션 선택적 생성

```
관리자
  ↓
입력 필드 채움 (여행지, 기간, 항공편, 출발지, 경비)
  ↓
섹션 토글 커스터마이징:
  ☑ 일반 안내사항
  ☑ 날씨
  ☑ 복장 / 준비물
  ☐ 환율 (언체크: 불필요)
  ☐ 수하물 정보
  ☐ 커스텀 공지
  ↓
[AI 안내문 생성]
  ↓
미리보기: 체크된 섹션만 표시
  ├─ 일반 안내사항 ✓
  ├─ 날씨 ✓
  ├─ 복장 / 준비물 ✓
  ├─ 환율 ✗ (생략)
  ├─ 수하물 정보 ✗ (생략)
  └─ 커스텀 공지 ✗ (생략)
  ↓
[저장] / [이미지 복사]
```

### 시나리오 4: 모바일 일정표 함께 전송

```
관리자
  ↓
[안내문 탭]
  ├─ 안내문 생성 및 미리보기 (P1 시나리오)
  └─ 하단: 모바일 일정표
    ↓
    날짜 | 항공편 | 도시 | 버스
    ─────────────────────────────
    12/18| AAL123| ICN→MNL | 출발
    12/18| AAL123| 비행 | -
    12/19| - | 마닐라 | 호텔→몰
    12/20| - | 마닐라 | 몰→해변
    12/22| AAL456| MNL→ICN | 비행
    ↓
[일정표 이미지 복사] 버튼
  ↓
html2canvas로 테이블 캡처 → 클립보드
  "일정표 이미지가 복사되었습니다."
  ↓
안내문 이미지 + 일정표 이미지 순서대로 카톡에 붙여넣기
```

  내용: "필리핀은 신용카드 사용 가능합니다. 환전은 공항에서 가능합니다."

## 에러 시나리오

### 에러 1: AI 생성 실패

```
관리자
  ↓
[AI 안내문 생성] 클릭
  ↓
Gemini API 호출 실패
  (네트워크 오류, API 키 오류, 토큰 초과 등)
  ↓
에러 메시지: "AI 생성에 실패했습니다. 나중에 다시 시도하세요."
  ↓
[다시 시도] 버튼 활성화
  ↓
입력 필드 유지, 다시 시도 가능
```

### 에러 2: 이미지 캡처 실패

```
관리자
  ↓
[안내문 이미지 복사] 클릭
  ↓
html2canvas 실패 (브라우저 호환성, 메모리 부족 등)
  ↓
에러 메시지: "이미지 캡처에 실패했습니다."
  ↓
회피책: 스크린샷 수동 이용 제안
```

### 에러 3: 클립보드 복사 실패

```
관리자
  ↓
[안내문 이미지 복사] 클릭
  ↓
이미지 생성 성공
  ↓
navigator.clipboard.write() 실패
  (권한 거부, HTTPS 아님 등)
  ↓
에러 메시지: "클립보드에 복사할 수 없습니다."
  ↓
회피책: [이미지 다운로드] 대체 버튼 제공
```

### 에러 4: 저장 실패

```
관리자
  ↓
[저장] 버튼 클릭
  ↓
POST /api/travel-guides 실패
  (DB 연결 오류, 데이터 검증 오류 등)
  ↓
에러 메시지: "저장에 실패했습니다. 네트워크 상태를 확인하세요."
  ↓
[다시 시도] 활성화, 입력 필드 유지
```

## 터치포인트 정리

| 단계 | 터치포인트 | 시간 | 액션 |
|------|----------|------|------|
| 1 | 항공편 선택 | 30초 | [항공편 불러오기] → 팝업 → 선택 |
| 2 | 데이터 자동 채움 | 즉시 | 여행지/기간/시간 자동 입력 |
| 3 | 추가 정보 입력 | 1분 | 출발지, 경비, 섹션 토글 |
| 4 | AI 생성 | 3~5초 | [AI 안내문 생성] → Gemini API |
