# Wireframe — 여행사 계약·견적·일정 자동화 시스템

## 1. 개요

본 문서는 여행사 인트라넷 시스템의 와이어프레임을 정의합니다.

### 1.1 와이어프레임 범례

```
┌─────┐  Border/Container
│     │
└─────┘

[Button]     버튼
[Input____]  입력 필드
○ Radio      라디오 버튼
☑ Checkbox   체크박스
```
