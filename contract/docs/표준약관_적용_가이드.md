# 표준약관 적용 가이드

## 목적
여행사 계약서를 공정거래위원회 또는 협회 표준약관에 맞게 수정하는 절차

## 1단계: 표준약관 문서 확보

### 다운로드 경로
1. **공정거래위원회 홈페이지** (www.ftc.go.kr)
   - 메뉴: 소비자 > 표준약관 > 여행
   - 국외여행 표준약관 / 국내여행 표준약관

2. **한국여행업협회** (www.kata.or.kr)
   - 자료실 > 표준계약서

3. **문화체육관광부** (www.mcst.go.kr)
   - 법령자료 > 표준계약서

### 필요한 문서
- [ ] 국외여행 표준약관 (해외여행용)
- [ ] 국내여행 표준약관 (국내여행용)
- [ ] 취소/환불 규정
- [ ] 안전정보 제공 조항

## 2단계: 현재 시스템 파일 위치

```
main/
└── api/
    └── templates/
        ├── base.html          # 공통 레이아웃
        ├── contract.html      # ⚠️ 계약서 템플릿 (수정 필요)
        ├── estimate.html      # 견적서 템플릿
        └── itinerary.html     # 일정표 템플릿
```

## 3단계: 템플릿 수정 방법

### 방법 A: 전체 교체 (권장)

1. 표준약관 Word/PDF 파일을 HTML로 변환
2. `api/templates/contract.html` 파일을 새 내용으로 교체
3. 동적 데이터 필드 추가 (단체명, 날짜 등)

### 방법 B: 조항별 수정

1. 현재 `contract.html` 열기
2. 표준약관과 비교하며 각 조항 수정
3. 필수 조항 누락 여부 확인

## 4단계: 필수 포함 조항 (공정거래위원회 기준)

### 국외여행 표준약관 필수 조항

- [ ] **제1조 (목적)**
- [ ] **제2조 (용어의 정의)**
- [ ] **제3조 (여행사의 의무)**
- [ ] **제4조 (여행자의 의무)**
- [ ] **제5조 (계약의 성립)**
- [ ] **제6조 (여행요금)**
- [ ] **제7조 (계약서 및 약관 교부 등)**
- [ ] **제8조 (여행사의 책임)**
- [ ] **제9조 (특별보상)**
- [ ] **제10조 (최저행사인원 미충족 시 계약해제)**
- [ ] **제11조 (여행출발 전 계약해제)**
- [ ] **제12조 (여행출발 후 계약해지)**
- [ ] **제13조 (여행요금의 변경)**
- [ ] **제14조 (손해배상)**
- [ ] **제15조 (여행사의 손해배상책임 제한)**
- [ ] **제16조 (안전정보 제공)**  ⚠️ **2019년 개정 - 필수**
- [ ] **제17조 (여행지에서의 안전수칙 준수)**
- [ ] **제18조 (개인정보 보호)**
- [ ] **제19조 (분쟁의 해결)**

### ⚠️ 중요 신규 조항 (2019년 개정)

**제16조 (안전정보 제공)**
```
여행사는 여행자에게 외교부 해외안전여행 사이트(www.0404.go.kr)에서
제공하는 여행지 안전정보를 제공하여야 한다.
```

## 5단계: 동적 데이터 필드 매핑

표준약관에 우리 시스템의 데이터를 삽입하는 부분:

### 필요한 데이터 필드

| 표준약관 항목 | 시스템 데이터 | Jinja2 변수 |
|------------|------------|------------|
| 여행자 (단체대표) | 단체명 | `{{ group.name }}` |
| 여행기간 | 출발일~도착일 | `{{ group.start_date }}` ~ `{{ group.end_date }}` |
| 여행일정 | 박수, 일수 | `{{ group.nights }}박 {{ group.days }}일` |
| 여행요금 | 총액 | `{{ group.total_price }}` |
| 계약금 | 선납금 | `{{ group.deposit }}` |
| 잔금 | 잔액 | `{{ group.balance }}` |
| 잔금완납일 | 잔액완납일 | `{{ group.balance_due_date }}` |
| 취소규정 | 취소규칙 목록 | `{{ cancel_rules }}` |
| 포함사항 | 포함/불포함 | `{{ includes }}` |
| 계약일 | 생성일 | `{{ generated_date }}` |

## 6단계: 작업 순서

1. ✅ **표준약관 다운로드** (위 경로에서)
2. ✅ **백업 생성**
   ```bash
   cd api/templates
   cp contract.html contract.html.backup
   ```
3. ✅ **새 계약서 작성**
   - 표준약관 내용 복사
   - HTML 형식으로 변환
   - Jinja2 변수 추가
4. ✅ **필수 조항 확인**
   - 제1조~제19조 모두 포함 여부
   - 특히 제16조 (안전정보) 필수
5. ✅ **테스트**
   - HTML 미리보기로 확인
   - 모든 데이터가 올바르게 표시되는지 검증
6. ✅ **법무 검토** (선택)
   - 회사 법무팀 또는 변호사 검토

## 8단계: 주의사항

### ⚠️ 법적 필수사항

1. **표준약관 번호 명시**
   - "공정거래위원회 제정 제10021호" 등 명시

2. **필수 조항 누락 금지**
   - 제1조~제19조 모두 포함 필수
   - 특히 제16조 (안전정보 제공) 필수

3. **임의 수정 금지**
   - 표준약관의 취지를 훼손하는 수정 불가
   - 여행자에게 불리한 변경 금지

4. **부속 약정서**
   - 표준약관 외 추가 조항은 "부속 약정서"로 별도 작성

## 9단계: 검증 체크리스트

- [ ] 공정거래위원회 표준약관 원문 확보
- [ ] 모든 조항 (제1조~제19조) 포함 확인
- [ ] 동적 데이터 필드 정상 작동 확인
- [ ] HTML 미리보기로 레이아웃 확인
- [ ] 인쇄/PDF 변환 테스트
- [ ] 법무팀 검토 (선택)
- [ ] 협회 회원사 확인 필요 시 협회 검토
