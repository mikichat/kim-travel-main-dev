# 새 단체 만들기 명단 미리보기 기능 테스트

## ✅ 추가된 기능

### 1. **명단 미리보기 테이블**
엑셀 파일을 업로드하면 모달 내에서 바로 명단을 확인할 수 있습니다.

#### 표시 항목:
- NO (번호)
- 한글 이름
- 영문 이름
- 성별
- 여권번호
- 생년월일

### 2. **자동 추출 + 명단 확인**
엑셀 파일 업로드 시:
1. ✅ 단체명 자동 추출 (예: "파타야 - 전라고 골프회")
2. ✅ 여행지역 자동 추출 (예: "파타야")
3. ✅ 출발일/귀국일 자동 추출 (예: "2025-12-17 ~ 2025-12-23")
4. ✅ **명단 미리보기 표시** (NEW!)

### 3. **사용자 경험 개선**
- 📋 명단이 있으면 즉시 테이블로 표시
- 📊 "✅ 8명 불러옴" 표시
- 🔄 스크롤 가능 (최대 400px 높이)
- 📌 헤더 고정 (스크롤해도 헤더는 보임)

## 🎯 작동 방식

### Before (이전):
```
엑셀 업로드 → "✅ 8명 불러옴" 표시만 → 생성 버튼 클릭
```

### After (현재):
```
엑셀 업로드
  ↓
단체명/날짜 자동 추출
  ↓
"✅ 8명 불러옴" 표시
  ↓
📋 명단 미리보기 테이블 표시 (NEW!)
  - 김숭기 (KIM/SOONGKEE) M M337FT026 1967-01-10
  - 김진우 (KIM/JIN WOO) M M281R7814 1967-06-29
  - ...
  ↓
확인 후 생성 버튼 클릭
```

## 📐 UI 변경사항

### 모달 크기 조정:
- **이전**: max-width: 500px
- **현재**: max-width: 800px (명단 테이블을 위해 확장)
- max-height: 90vh (화면 높이에 맞춤)
- overflow-y: auto (긴 내용도 스크롤 가능)

### 테이블 스타일:
- 최소 너비: 600px
- 수평/수직 스크롤 모두 지원
- Sticky 헤더로 스크롤 시 헤더 고정
- 얼룩말 무늬 (행마다 배경색 다름)

## 🧪 테스트 방법

1. 단체 명단 관리 페이지 열기
2. "➕ 새 단체 만들기" 버튼 클릭
3. "📁 엑셀 가져오기" 버튼 클릭
4. 엑셀 파일 선택 (예: "명단 -251217-0.xlsx")
5. ✅ 자동으로 다음이 채워짐:
   - 단체명: "파타야 - 전라고 골프회"
   - 여행지역: "파타야"
   - 출발일: 2025-12-17
   - 귀국일: 2025-12-23
   - **명단 미리보기 테이블** (NEW!)
6. 명단 확인 후 "생성" 버튼 클릭

## 💡 장점

1. **투명성**: 어떤 명단이 추가될지 미리 확인 가능
2. **정확성**: 생성 전에 데이터 검증 가능
3. **편의성**: 명단을 확인하기 위해 다른 페이지로 이동할 필요 없음
4. **효율성**: 한 화면에서 모든 정보 확인 가능

## 🎨 디자인

- 깔끔한 테이블 레이아웃
- 고정 헤더로 스크롤 시 항상 컬럼명 표시
- 테두리와 그림자로 명확한 구분
- 작은 폰트(13px)로 많은 정보 표시
- 적절한 패딩으로 가독성 확보
