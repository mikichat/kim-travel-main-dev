# 골프/혼합 일정표 디자인 개선 제안서

> 작성일: 2026-03-06
> 대상: ItineraryLuxury 컴포넌트의 golf / mixed dayType 렌더링

---

## 1. 현재 상태 분석

### 1.1 현재 골프 블록 구조
```
+-----------------------------------------------+
| [연한 녹색 배경 #d8f3dc, 테두리 #52b788]         |
|                                                 |
| ⛳ 호이아나 C.C. Hoiana Shores Golf Club        |  <- 코스명
|                                                 |
| 한국어 코스 설명 3~7줄...                        |  <- description
|                                                 |
| 골프조 [2조 8명]                                 |  <- groupInfo
| ▶ 18홀 라운딩 - 티업시간: 07:20                  |  <- holes + teeTime
| - 그린피 & 카트 & 캐디피 포함                     |  <- includedItems
| - 캐디팁(20$/1인/18홀) 불포함                     |  <- excludedItems (빨간)
|                                                 |
| 관광조(3인) - 호텔 자유시간                       |  <- mixed 전용
+-----------------------------------------------+
```

### 1.2 현재 문제점

| 문제 | 설명 |
|------|------|
| **단조로운 레이아웃** | 텍스트만 나열, 시각적 계층 부족 |
| **골프/관광 구분 약함** | mixed일에 골프조/관광조가 같은 블록 안에 대시선 하나로 구분 |
| **코스 정보 부족** | 설계자, 야드, 파수 등 프리미엄 정보 없음 |
| **아이콘 단일** | ⛳ 하나만 사용, 시각적 풍부함 부족 |
| **이미지 미활용** | 골프장 이미지가 있어도 블록 내부에서 활용되지 않음 |

## 2. 경쟁사 디자인 패턴 분석

### 2.1 한국 여행사 패턴
- **하나투어/모두투어**: 상품명에 홀수 명시 ("54홀"), 일정표에 코스별 설명 블록 삽입
- **럭셔리골프투어**: 홀수 숫자열 표기 `0-18-18-18-0`, 포함사항 3단 구조
- **파로스골프**: 설계자명 + 야드수 + 파수 명시 ("Greg Norman 설계, 7,008야드, 파72")

### 2.2 해외 여행사 패턴
- **Golfasian**: 골퍼/비골퍼(non-golfer) 병렬 트랙으로 같은 날 두 일정 표현
- **PerryGolf**: 포함사항 세분화 (그린피, 카트, 캐디, 장비운송 각각 명시)
- **Golf Travel Design**: 인터랙티브 웹 브로슈어, 모바일 대응

### 2.3 색상 업계 표준
- 메인: 골프 그린 `#2d6a4f` 계열
- 액센트: 골드 `#F3B31A` (프리미엄 느낌)
- 현재 사용 중인 `#d8f3dc` / `#52b788` / `#2d6a4f` 조합은 업계 표준에 부합

## 3. 개선 제안

### 제안 A: 골프 블록 내부 리디자인 (권장, 소규모)

기존 단일 블록 구조를 유지하되 **내부 레이아웃을 개선**하는 방안.
코드 변경 최소화, 인쇄 호환성 유지.

#### Before → After 비교

```
[Before]                              [After]
+---------------------------+         +------------------------------------------+
| ⛳ 호이아나 C.C.           |         | ⛳ 호이아나 C.C.                          |
|                            |         |    Hoiana Shores Golf Club                |
| 코스 설명 텍스트...        |         |                                          |
|                            |         | 코스 설명 텍스트...                       |
| 골프조 [2조 8명]           |         |                                          |
| ▶ 18홀 라운딩 - 07:20     |         | +--------------------------------------+ |
| - 포함: ...               |         | | 18홀  |  티타임   |  포함     | 불포함 | |
| - 불포함: ...             |         | | PAR72 |  07:20   | 그린피   | 캐디팁 | |
|                            |         | |       |          | 카트     |        | |
| 관광조(3인) - 자유시간     |         | |       |          | 캐디피   |        | |
+---------------------------+         | +--------------------------------------+ |
                                      |                                          |
                                      | 골프조 [2조 8명]                          |
                                      +------------------------------------------+
```

#### 핵심 변경 사항

**3-A-1. 라운딩 정보를 미니 테이블로 변경**
```
현재: 텍스트 나열 (▶ 18홀 라운딩 - 티업시간: 07:20)
개선: 4칸 미니 그리드
      | 홀수    | 티타임  | 포함사항  | 불포함   |
      | 18홀    | 07:20  | 그린피    | 캐디팁   |
      | PAR 72  |        | 카트      | 20$/인   |
```

**3-A-2. 코스명 영문 표기 개선**
```
현재: ⛳ 호이아나 C.C. Hoiana Shores Golf Club (한줄)
개선: ⛳ 호이아나 C.C.
        Hoiana Shores Golf Club      (2줄, 영문 작게)
```

**3-A-3. mixed 관광조 시각적 분리 강화**
```
현재: ---대시 테두리--- 관광조(3인) - 호텔 자유시간
개선: 별도 색상 배경 (#f0f4ff 연한 파랑) + 카메라 아이콘
      [카메라] 관광조(3인) — 호텔 자유시간
```

### 제안 B: 골프 데이 카드 전체 디자인 변경 (중규모)

골프일의 dayCard 자체를 관광일과 시각적으로 차별화.

#### 핵심 변경 사항

**3-B-1. 좌측 dayBar 색상 변경**
```
관광일: 기존 색상 (홀수 #e67e22 / 짝수 #5bc0de)
골프일: 골프 그린 (#2d6a4f)
혼합일: 그린 + 오렌지 그라데이션
```

**3-B-2. dayType 라벨 배지**
```
일차 숫자 옆에 작은 배지 표시:
  [2일차] GOLF     ← 초록 배지
  [3일차] MIXED    ← 초록+파랑 배지
  [4일차]          ← 관광일은 배지 없음
```

**3-B-3. 골프일 상단 요약 바**
```
+--------------------------------------------------+
| ⛳ 호이아나 C.C. | 18홀 | 티타임 07:20 | PAR 72  |
+--------------------------------------------------+
| 코스 설명...                                      |
| 세부 일정...                                      |
```
일정표 스타일의 meta 영역처럼, 골프 핵심 정보를 한 줄 요약으로 먼저 보여주는 방식.

---

### 제안 C: 홀수 숫자열 + 골프 요약 헤더 (대규모, 장기)

일정표 전체 상단에 골프 요약 정보를 추가하는 방안.

#### 일정표 타이틀 아래 골프 요약 섹션
```
+--------------------------------------------------+
|        베트남 다낭 골프 3박 5일                     |
|        2026.03.15 ~ 2026.03.19                    |
+--------------------------------------------------+
| 라운딩 스케줄: 0 - 18 - 18 - 18 - 0  (총 54홀)    |
| Day 2: 호이아나 C.C. | Day 3: 바나힐스 G.C.       |
| Day 4: 몽고메리 링크스                             |
+--------------------------------------------------+
```

## 6. 타입 확장 (선택)

현재 `GolfRound` 인터페이스에 추가 가능한 필드:

```typescript
interface GolfRound {
    // 기존 필드 유지
    courseName: string;
    courseNameKo: string;
    description: string;
    teeTime: string;
    holes: number;
    includedItems: string;
    excludedItems: string;
    groupInfo: string;
    sightseeingGroupInfo: string;
    images: string[];

    // 추가 제안 필드
    par?: number;           // 파 수 (예: 72)
    yardage?: number;       // 야드 (예: 7008)
    designer?: string;      // 설계자 (예: "Robert Trent Jones Jr.")
}
```

이 필드들은 선택적이므로 기존 데이터와 호환됩니다.
TourForm에서 입력 UI를 추가하면 됩니다.
