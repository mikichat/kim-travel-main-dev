# User Flow
## 여행사 관리 시스템 핵심 업무 흐름

### 1. 신규 예약 생성 및 관리 (Booking Flow)

1.  **예약 관리 페이지 이동**
    *   사이드바 > '예약 관리' 클릭.
2.  **새 예약 추가**
    *   '새 예약 추가' 버튼 클릭.
    *   예약 정보 입력 폼 활성화.
3.  **정보 입력**
    *   고객명, 상품명, 출발일, 인원 등 필수 정보 입력.
    *   초기 상태 설정 (예: '문의').
4.  **저장**
    *   '저장' 버튼 클릭 -> LocalStorage에 저장됨.
    *   목록 갱신 및 대시보드 통계 반영.

### 2. 견적서 생성 (Quote Generation Flow)

1.  **견적서 생성 페이지 이동**
    *   사이드바 > '견적서 생성' 클릭.
2.  **고객 및 상품 선택**
    *   '기존 고객 선택' 또는 '직접 입력' 선택.
    *   상품 선택 시 기본 정보 로드.
3.  **상세 조건 입력**
    *   항공사, 호텔, 출발/도착 시간, 추가 금액 등 입력.
4.  **미리보기 및 출력**
    *   '미리보기' 버튼으로 견적서 레이아웃 확인.
    *   '견적서 생성 (인쇄)' 버튼으로 PDF 저장 또는 인쇄.

### 3. 단체명단 관리 (Group Roster Flow)

1.  **상품 관리 > 단체 연동**
    *   특정 여행 상품에 고객들을 배정.
2.  **단체명단 페이지 이동**
    *   사이드바 > '단체명단' 클릭.
3.  **명단 확인 및 엑셀 작업**
    *   배정된 고객 리스트 확인.
    *   '엑셀 내보내기'를 통해 항공사/투어사 제출용 명단 생성.

### 4. 일정표 발송 (Itinerary Flow)

1.  **일정표 메뉴 이동**
    *   사이드바 > 일정표 > 'PDF 일정표 생성' 또는 '상세/간략 일정' 선택.
2.  **데이터 입력/로드**
    *   여행 기간, 항공편, 일자별 스케줄 입력.
3.  **출력**
    *   최종 일정표 확인 후 클립보드 복사 또는 파일 저장하여 고객에게 발송.
