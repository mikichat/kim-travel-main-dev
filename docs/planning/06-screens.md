# 화면 설계서 (Screen Design & Component Specification)

**문서 버전**: 1.0 | **작성일**: 2026-02-18

---

## MVP 캡슐

| 항목 | 내용 |
|------|------|
| **에디터 페이지** | hanatour/travel-free.html (입력) |

## 1. 에디터 페이지 (travel-free.html)

### 레이아웃 구조

```
┌─────────────────────────────────────┐
│ 사이드바 (기존 style.css)            │
│                                     │ Main Content Area
│                                     │ ┌───────────────────────────┐
│ - 로고                              │ │ 헤더: 저장/불러오기 버튼  │
│ - 메뉴 (일정표 > 자유여행 예약)      │ ├───────────────────────────┤
│ - 기타                              │ │                           │
│                                     │ │ [1] 기본정보 섹션        │
│                                     │ │ ☐ 수신인: ___________    │
│                                     │ │ ☐ 발신인: ___________    │
│                                     │ │ ☐ 작성일: YYYY-MM-DD    │
│                                     │ │                           │
│                                     │ │ [2] 항공편 섹션          │
│                                     │ │ ☐ [카드 1]               │
│                                     │ │    탑승객 x 2명          │
│                                     │ │    KE101 (2025-01-28)   │
│                                     │ │ ☐ [+] 추가 버튼         │
│                                     │ │                           │
│                                     │ │ 실시간 미리보기 패널     │
│                                     │ │                           │
│                                     │ └───────────────────────────┘
└─────────────────────────────────────┘
```

### 주요 컴포넌트

#### 1. 헤더 섹션
- 저장 버튼 (localStorage에 저장)
- 불러오기 버튼 (저장 목록에서 선택)
- 미리보기 버튼 (새 탭에서 preview-free.html 열기)

#### 2. 기본정보 섹션

## 2. 미리보기 페이지 (preview-free.html)

### 레이아웃 구조

```
┌─────────────────────────┐
│ 상단 바: 돌아가기 | 저장 | 인쇄
├─────────────────────────┤
│ [여행세상 로고]          │
│ 예약 내역               │
│ 수신인: 이귀운          │
│ 발신인: 여행세상        │
│ 작성일: 2025-01-06     │
├─────────────────────────┤
│ 항공편 섹션             │
│ ┌─────────────────────┐ │
│ │ ✈ KE101           │ │
│ │ 인천 → 제주        │ │
│ │ 탑승객: 김귀진, 미진 │ │
│ └─────────────────────┘ │
├─────────────────────────┤
│ 숙박 섹션               │
│ ┌─────────────────────┐ │
│ │ 🏨 신라 제주        │ │
│ │ 2025-01-28 3박     │ │
│ │ 예약자: 김귀진      │ │
│ └─────────────────────┘ │
├─────────────────────────┤
│ 렌트카, 골프, 커스텀... │
├─────────────────────────┤
│ 결제 정보 카드          │
│ (파란 그라데이션 배경)  │
├─────────────────────────┤
│ 회사정보 + 도장         │
├─────────────────────────┤
│ 하단 바: 저장 | 인쇄    │
└─────────────────────────┘
```

### 주요 컴포넌트

#### 상단 네비게이션
- 돌아가기 (에디터로)
- 이미지 저장 (전체 또는 섹션별)
- 인쇄
- 다크모드 토글

#### 카드 UI (각 예약 항목)
- 카테고리별 컬러 배지
- 아이콘 (Material Symbols)
- 핵심 정보 한눈에
- 타이핑 애니메이션 (선택)

#### 결제 정보 (특별 스타일)
- 파란 그라데이션 배경 (blue → purple)
- 큰 금액 표시
- 하얀 텍스트

#### 회사정보 (푸터)
- 회사명, 대표, 주소
- 연락처
- 도장 이미지

#### 하단 제어 바
- 저장 (이미지)
- 인쇄
- 문의하기 (tel: 링크)

## 3. 컴포넌트 재사용 패턴

### 섹션 카드 (재사용)
```html
<div class="section-card">
  <div class="card-icon">아이콘</div>
  <div class="card-title">제목</div>
  <div class="card-content">내용</div>
  <div class="card-meta">메타정보</div>
</div>
```

### 입력 필드 (재사용)
```html
<div class="form-group">
  <label>라벨</label>
  <input type="text" class="form-input" />
</div>
```

### 버튼 (재사용)
```html
<button class="btn btn-primary">확인</button>
<button class="btn btn-secondary">취소</button>
<button class="btn btn-danger">삭제</button>
<button class="btn btn-sm">작은 버튼</button>
```

### 모달 (저장/불러오기)
```html
<div class="modal" id="saveModal">
  <div class="modal-content">
    <h3>저장 이름</h3>
    <input type="text" placeholder="저장 이름 입력" />
    <button class="btn btn-primary">저장</button>
    <button class="btn btn-secondary">취소</button>
  </div>
</div>
```

## 4. 상호작용 명세

| 요소 | 트리거 | 동작 |
|------|--------|------|
| 추가 버튼 | 클릭 | 새 입력 폼 표시 |
| 삭제 버튼 | 클릭 | 확인 후 항목 제거 |
| 섹션 토글 | 체크박스 | 미리보기 업데이트 |
| 저장 버튼 | 클릭 | localStorage 저장 |
| 불러오기 | 선택 | 데이터 로드 |
| 이미지 저장 | 클릭 | html2canvas 실행 |
| 인쇄 | 클릭 | window.print() |

## 5. 반응형 브레이크포인트

| 기기 | 너비 | 레이아웃 |
|------|------|---------|
| 모바일 | < 640px | 한 칼럼 |
| 태블릿 | 640-1024px | 두 칼럼 |
| 데스크톱 | > 1024px | 세 칼럼 + 사이드바 |
