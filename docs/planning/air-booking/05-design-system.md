# 항공 예약 관리 시스템 디자인 시스템

## 1. 디자인 원칙

- **눈 편한 UI**: 큰 글씨, 높은 대비, 넉넉한 여백 — 장시간 업무 시력 배려
- **색상으로 소통**: 빨강(긴급)/노랑(임박)/초록(완료) — 상태를 한눈에
- **업무 중심**: 불필요한 장식 최소화, 정보 밀도 적정

| 용도 | 크기 | 굵기 | 비고 |
|------|------|------|------|
| H1 (페이지 타이틀) | 28px | Bold | |

**폰트**: Pretendard (한글 + 영문 통합)

## 4. 컴포넌트

### Button

| 타입 | 배경 | 텍스트 | 용도 |
|------|------|--------|------|
| Primary | #2563EB | white | 주요 액션 (저장, 등록) |
| Secondary | transparent | #2563EB | 보조 액션 |
| Danger | #DC2626 | white | 삭제, 취소 |
| Ghost | transparent | #64748B | 부가 액션 |

- **크기**: 최소 높이 44px (터치 타겟)
- **패딩**: 12px 24px

### Input

| 상태 | 테두리 | 배경 |
|------|--------|------|
| Default | #E2E8F0 | white |
| Focus | #2563EB | white |
| Error | #DC2626 | #FEF2F2 |
| Disabled | #E2E8F0 | #F8FAFC |

- **높이**: 44px
- **폰트 크기**: 16px (자동 줌 방지)

### Table

- **행 높이**: 최소 52px (큰 글씨 모드 시 60px)
- **확장 행**: 패딩 16px, 배경 #F8FAFC
- **호버**: 배경 #F1F5F9
- **상태 뱃지**: 왼쪽 컬러 바 4px

### Calendar

- **날짜 셀**: 최소 80x80px
- **이벤트 마커**: 점(dot) + 색상 (빨강/노랑/초록)
- **오늘**: 파란색 테두리

### Sidebar

- **너비**: 240px (축소 시 64px)
- **아이콘 + 텍스트** 조합
- **활성 항목**: 파란색 배경

## 6. 반응형

| 브레이크포인트 | 값 | 대상 |
|-------------|-----|------|
| Desktop | > 1024px | 기본 (사무실 PC) |
| Tablet | 768px ~ 1024px | 참고용 |

**주 타겟**: 데스크톱 (사무실 PC 모니터)

## 7. 접근성

- **큰 글씨 모드**: 본문 16px → 18px, 제목 비례 증가
- **다크모드**: 토글로 전환
- **대비율**: WCAG AA 이상 (4.5:1)
- **포커스 표시**: 모든 인터랙티브 요소에 포커스 링
