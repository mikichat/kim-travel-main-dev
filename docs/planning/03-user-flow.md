# 사용자 흐름도 (User Flow)

**문서 버전**: 1.0 | **작성일**: 2026-02-18

---

## MVP 캡슐

| 항목 | 내용 |
|------|------|
| **주요 흐름** | 여행사 입력 → 저장 → 미리보기 → 고객 공유 |
| **고객 흐름** | URL 링크 → 열람 → 이미지 저장 |
| **핵심 분기** | 섹션 토글, 저장/불러오기 |

---

## 1. 여행사 직원 흐름

### 메인 흐름 (Happy Path)

```
시작
  ↓
[에디터 페이지 접근]
  ↓
[기본정보 입력]
  (수신인, 발신인, 작성일, 여행지, 여행기간)
  ↓
[항공편 추가] (선택)
  (탑승객, 편명, 날짜, 출발/도착 공항/시간)
  ↓
[숙박 추가] (선택)
  (숙소명, 체크인, 박수, 예약자)
  ↓
[렌트카 추가] (선택)
  (차종, 보험, 예약자)
  ↓
[골프 추가] (선택)
  (골프장, 날짜, 티오프, 예약자)
  ↓
[커스텀 섹션 추가] (선택)
  (타입, 항목명, 날짜, 메모, 예약자)
  ↓
[결제정보 입력] (선택)
  (총액, 은행, 계좌, 예금주, 상태)
  ↓
[회사정보 입력] (선택)
  (회사명, 대표, 주소, 전화, 팩스, 도장, 담당자)
  ↓
[미리보기 확인]
  - 섹션별 토글 (표시/숨김)
  - 실시간 미리보기 패널 확인
  ↓
[저장 또는 미리보기 페이지 열기]
  ├─ Save: localStorage 저장 (자동 저장됨)
  └─ Preview: 새 탭에서 미리보기 페이지 열기
  ↓
종료 (또는 다른 고객 예약 시작)
```

### 대체 흐름: 저장/불러오기

```
[기존 작업 불러오기]
  ↓
[저장 목록에서 선택]
  ↓
[에디터에 데이터 로드]
  ↓
[계속 편집 또는 저장]
```

### 대체 흐름: 미리보기에서 공유

```
[미리보기 페이지]
  ↓
[공유 버튼 클릭]
  ├─ 이미지 저장: html2canvas → PNG
  └─ URL 공유: LZString → query parameter
  ↓
[고객에게 전달]
  ├─ 이메일 첨부 (이미지)
  └─ URL 링크 (카카오톡, SMS)
```

## 2. 자유여행 고객 흐름

### 메인 흐름

```
[여행사에서 받은 URL 링크 클릭]
  ↓
[preview-free.html?data=... 접근]
  ↓
[LZString 자동 해제]
  ↓
[예약 내역 자동 로드]
  ↓
[미리보기 페이지 표시]
  - 기본정보
  - 항공편 카드
  - 숙박 카드
  - 렌트카 카드
  - 골프 카드
  - 커스텀 섹션
  - 결제 정보
  - 회사정보
  ↓
[이미지 저장 또는 인쇄]
  ├─ 전체 이미지 저장: html2canvas
  ├─ 섹션별 이미지 저장: 체크박스 선택
  └─ 인쇄: window.print()
  ↓
종료
```

### 대체 흐름: 이메일 첨부 이미지 확인

```
[여행사에서 받은 이메일]
  ↓
[첨부 이미지 확인]
  ↓
[모바일 갤러리에 저장]
```

## 3. 주요 상호작용 (Interactions)

### 에디터 페이지

| 작업 | 트리거 | 결과 |
|------|--------|------|
| 기본정보 입력 | 텍스트 입력 | 실시간 저장 (5초) |
| 항공편 추가 | "추가" 버튼 | 새 입력 폼 표시 |
| 항공편 삭제 | "삭제" 버튼 | 항목 제거 + 미리보기 반영 |
| 섹션 토글 | 체크박스 | 미리보기에서 표시/숨김 |
| 저장 | "저장" 버튼 | localStorage에 저장 + 목록 업데이트 |
| 불러오기 | 저장 목록 선택 | 에디터에 데이터 로드 |
| 미리보기 | "미리보기" 버튼 | 새 탭에서 preview-free.html 열기 |

### 미리보기 페이지

| 작업 | 트리거 | 결과 |
|------|--------|------|
| 이미지 저장 | "저장" 버튼 | html2canvas → PNG 다운로드 |
| 섹션별 저장 | 섹션 체크박스 | 선택된 섹션만 이미지화 |
| 인쇄 | "인쇄" 버튼 | window.print() |
| 돌아가기 | "뒤로" 버튼 | 에디터 페이지로 이동 |
| 다크모드 | 토글 버튼 | 테마 전환 + localStorage 저장 |

## 4. 데이터 흐름

### 저장 경로

```
에디터 입력
  ↓
5초 타이머 (디바운싱)
  ↓
JSON 직렬화
  ↓
localStorage.setItem('freetravel_saves_v1', JSON.stringify(data))
  ↓
저장 완료 피드백 (선택)
```

### 공유 경로

```
에디터 데이터
  ↓
JSON.stringify(data)
  ↓
LZString.compressToBase64(json)
  ↓
const url = `preview-free.html?data=${compressed}`
  ↓
고객에게 URL 링크 전달
  ↓
고객 URL 접근
  ↓
LZString.decompressFromBase64(urlParams.data)
  ↓
JSON.parse(decompressed)
  ↓
미리보기 페이지 렌더링
```

## 5. 조건부 흐름

### 항목 추가/삭제 로직

```
"추가" 버튼 클릭
  ↓
새 입력 폼 표시 (맨 아래)
  ↓
사용자 입력
  ↓
"확인" 클릭
  ↓
배열에 아이템 추가
  ↓
미리보기 자동 업데이트
```

### 섹션 토글 로직

```
섹션 체크박스 ON/OFF
  ↓
sections 객체 업데이트
  ↓
미리보기 페이지에서 섹션 표시/숨김
  ↓
저장 (자동)
```

### 이미지 저장 옵션 (Phase 2)

```
사용자가 "이미지 저장" 클릭
  ↓
[모달] 전체 vs 섹션별 선택
  ├─ 전체: html2canvas('.preview-container')
  └─ 섹션별: querySelectorAll('[data-section]')로 각각 저장
  ↓
PNG 파일 다운로드
```

## 6. 에러 처리 흐름

### localStorage 저장 실패

```
try {
  localStorage.setItem(...)
} catch (e) {
  if (e.name === 'QuotaExceededError') {
    [알림] "저장 공간이 부족합니다"
    [옵션] 이전 저장 삭제
  }
}
```

### 이미지 생성 실패

```
html2canvas(element)
  .catch(error => {
    [알림] "이미지 저장에 실패했습니다"
    [옵션] 섹션별 저장 시도
  })
```

### 잘못된 URL 접근

```
urlParams.data 없음
  ↓
[알림] "유효하지 않은 링크입니다"
  ↓
홈페이지로 리다이렉트
```

## 7. 성능 최적화 흐름

### 자동 저장 디바운싱

```
사용자 입력
  ↓
타이머 초기화 (500ms)
  ↓
사용자 입력 계속?
  ├─ Yes: 타이머 재설정
  └─ No: 500ms 후 저장
```

### 미리보기 렌더링

```
데이터 변경
  ↓
미리보기 섹션만 업데이트 (전체 리렌더 아님)
  ↓
CSS transition으로 부드러운 애니메이션
```
