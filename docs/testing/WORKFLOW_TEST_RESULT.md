## 📋 테스트 개요

### 테스트 목적
항공편 변환기 → 스케줄 관리 → 견적서 → 일정표의 전체 워크플로우가 정상적으로 작동하는지 검증

### 테스트 범위
1. ✈️ 항공편 변환기(air1): 항공편 입력 및 저장
2. 📅 스케줄 관리(flight-schedule): 실시간 동기화
3. 📋 견적서 편집기(quote-editor-v1): 항공편 선택 및 자동 입력
4. 📄 일정표(travel-simple): 데이터 선택 및 자동 입력

### ✅ 1단계: 항공편 변환기 (air1/index.html)

**테스트 항목**:
- [ ] 페이지 로드 확인
- [ ] 항공편 정보 입력

- [ ] localStorage 저장 확인
- [ ] 저장 성공 메시지 확인

**입력 데이터**:
```
[항공편 정보]
- PNR: OSK20250210
- 항공편 1 (출발):
  * 편명: OZ 111
  * 날짜: 2025.02.10(월)

  * 편명: OZ 112
  * 날짜: 2025.02.14(금)
  * 출발: 간사이공항 (KIX) 12:30
  * 도착: 인천국제공항 (ICN) 14:30

[고객 정보]
- 대표자: 김여행
- 연락처: 010-1234-5678
- 인원: 32명

[미팅 정보]
- 장소: 인천공항 제1터미널 3층 J카운터
- 시간: 2025.02.10 07:00
```
