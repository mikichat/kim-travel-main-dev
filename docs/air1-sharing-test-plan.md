## 2. Phase 1 테스트: 클립보드 복사

### 2.1 이미지 복사 (단체/거래처)

| # | 테스트 | 입력 | 기대 결과 |
|---|--------|------|----------|
| T1 | 이미지 클립보드 복사 (Chrome) | 단체 PNR 변환 → [이미지 복사] | 토스트 "클립보드에 복사됨", 카톡 PC Ctrl+V 가능 |
| T2 | 거래처 포함 복사 | 거래처 "평화관광" 입력 → [이미지 복사] | 캡처 이미지에 거래처명 포함 |
| T3 | 승객명단 포함 복사 | 전체 승객 체크 → [이미지 복사] | 3열 승객명단 포함 |
| T4 | 고객정보 미포함 (단체) | 단체 PNR → [이미지 복사] | 이미지에 고객정보 섹션 없음 |
| T5 | 고객정보 폼 미캡처 | [이미지 복사] | 체크박스/입력폼 캡처 안 됨 |
| T6 | 폴백 (Firefox) | Firefox에서 [이미지 복사] | PNG 다운로드 + 토스트 |
| T7 | 변환 전 클릭 | 변환 안 한 상태 → [이미지 복사] | 반응 없음 (outputSection hidden) |

### 2.2 모바일 복사 (개인 고객)

| # | 테스트 | 입력 | 기대 결과 |
|---|--------|------|----------|
| T8 | 모바일 클립보드 복사 | 개인 PNR + 고객정보 → [모바일 복사] | 토스트 "클립보드에 복사됨" |
| T9 | 고객정보 포함 확인 | 이름/전화/미팅 입력 → Ctrl+V | 카드형 이미지에 고객정보 표시 |
| T10 | 가격 포함 확인 | 성인 1,250,000원 → [모바일 복사] | 카드에 가격 표시 |
| T11 | 거래처 포함 | 거래처 입력 → [모바일 복사] | 카드에 거래처 표시 |
| T12 | 변환 전 클릭 | 변환 안 한 상태 → [모바일 복사] | 경고 토스트 |
| T13 | iframe 렌더링 | [모바일 복사] | 폰트 로드 완료, 카드 깨짐 없음 |

### 2.3 카톡 전송 End-to-End

| # | 테스트 | 기대 결과 |
|---|--------|----------|
| T14 | 이미지 복사 → 카톡 PC 붙여넣기 | Ctrl+V로 이미지 전송됨 |
| T15 | 모바일 복사 → 카톡 PC 붙여넣기 | Ctrl+V로 카드형 이미지 전송됨 |
| T16 | 붙여넣기 이미지 선명도 | 텍스트 읽기 가능, 이모지 정상 |
| T17 | 한글 깨짐 없음 | 한글 이름/지명 정상 표시 |

## 7. 수동 테스트 체크리스트 (Phase 1 릴리스 전)

- [ ] Chrome localhost:5000/air1 접속
- [ ] 로그인 상태 → 직원 모드 활성화 확인
- [ ] 단체 PNR 변환 → [이미지 복사] → 카톡 PC Ctrl+V → 이미지 전송 확인
- [ ] 개인 PNR 변환 → 고객정보 입력 → [모바일 복사] → 카톡 PC Ctrl+V → 이미지 전송 확인
- [ ] 거래처 "평화관광" 입력 → 이미지에 포함 확인
- [ ] 카톡에서 수신한 이미지 한글 깨짐 없음 확인
- [ ] 콘솔 에러 없음 확인
- [ ] Firefox에서 폴백 (다운로드) 정상 확인
