# PRD: 자유여행 예약내역 페이지

## 1. 개요

### 1.1 목적
자유여행 고객에게 예약 내역(항공, 호텔, 렌터카, 골프 등)을 모바일 최적화된 페이지로 전달하는 시스템.
기존 `hanatour/travel-simple.html`의 **에디터 + 미리보기** 패턴을 재활용한다.

### 1.2 핵심 가치
- 여행사 직원이 에디터에서 예약 내역을 입력
- 모바일 최적화 미리보기 페이지를 고객에게 링크로 공유
- 섹션별 "미리보기 포함" 체크박스로 노출 항목 선택

### 1.3 사용자
| 사용자 | 역할 |
|--------|------|
| 여행사 직원 (관리자) | 에디터에서 예약 내역 입력/편집/저장 |
| 고객 (수신자) | 모바일 미리보기 페이지에서 예약 내역 확인 |

## 2. 기능 요구사항

### 2.1 에디터 페이지 (`hanatour/travel-free.html`)

#### 2.1.1 기본 정보 섹션
| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 수신인 | text | Y | 고객명 (예: "이귀운 권사님") |
| 발신인 | text | Y | 기본값: "여행세상" |
| 작성일 | date | Y | 기본값: 오늘 날짜 |
| 여행 기간 | date range | N | 출발일 ~ 귀국일 |
| 여행지 | text | N | 예: "제주도", "오사카" |

#### 2.1.2 동적 섹션 (각각 "미리보기 포함" 체크박스)

**항공편** `[✓ 미리보기 포함]`
- 탑승객 그룹 (이름 태그 + 탑승객 번호)
- 항공편 카드 (편명, 날짜, 출발공항/시간, 도착공항/시간)
- 항공편 추가/삭제 버튼
- 탑승객 그룹 추가/삭제 버튼

**숙박** `[✓ 미리보기 포함]`
- 숙소명, 체크인 날짜, 박수, 예약자명
- 숙소 추가/삭제 버튼

**렌터카** `[✓ 미리보기 포함]`
- 차종, 예약자명, 보험 정보
- 렌터카 추가/삭제 버튼

**골프** `[✓ 미리보기 포함]`
- 골프장명, 날짜, 티오프 시간, 예약자명
- 골프 추가/삭제 버튼

**커스텀 섹션** `[✓ 미리보기 포함]`
- 섹션 타이틀 (셀렉트: 투어/식사/입장권/액티비티/기타)
- 아이콘 자동 매칭
- 항목명, 날짜, 메모, 예약자
- 커스텀 섹션 추가/삭제 버튼

**결제 정보** `[✓ 미리보기 포함]`
- 총 결제 금액 (숫자 → 원화 포맷 자동)
- 은행명, 계좌번호, 예금주
- 결제 상태 (미결제/부분결제/결제완료)

**회사 정보** `[✓ 미리보기 포함]`
- 회사명, 대표자명, 주소, 전화, 팩스
- 도장 이미지 업로드
- 담당자 이름/연락처

#### 2.1.3 저장/불러오기
- localStorage 기반 (`freetravel_saves_v1`)
- 저장 목록 모달 (이름 지정)
- 불러오기 모달 (목록에서 선택)

#### 2.1.4 미리보기 생성
- "미리보기 & 공유하기" 버튼
- LZString 압축 → URL 파라미터 → 새 탭 열기
- 이미지는 localStorage(`__preview_images_free__`)에 별도 저장

### 2.2 미리보기 페이지 (`hanatour/preview-free.html`)

#### 2.2.1 레이아웃
- 모바일 최적화 (`max-w-md mx-auto`)
- 사이드바 없음 (고객용 페이지)
- 스택형 카드 UI (rounded-2xl, shadow, border)
- 다크모드 지원

#### 2.2.2 상단 고정 바
- 뒤로가기, 링크 복사, 이미지 저장(html2canvas), 인쇄

#### 2.2.3 콘텐츠 섹션 (에디터에서 체크된 항목만 노출)
1. **헤더**: 여행세상 로고, "예약 내역" 타이틀, 수신인/발신인/작성일
2. **항공편 카드**: 탑승객 아바타, 편명 배지, 출발↔도착 시각 (비행기 아이콘 라인)
3. **숙박 카드**: 호텔 아이콘, 숙소명, 날짜(박수), 예약자
4. **렌터카 카드**: 차 아이콘, 차종, 보험 배지, 예약자
5. **골프 카드**: 골프 아이콘, 골프장명, 날짜, 티오프 시간
6. **커스텀 카드**: 동적 아이콘, 항목 리스트
7. **결제 카드**: 파란 배경, 총액 대형 폰트, 계좌 정보
8. **푸터**: 회사 정보, 도장 이미지, 문의처

#### 2.2.4 하단 고정 바
- 상세내역(스크롤 top), 저장(이미지), 인쇄, 문의하기(전화 링크)

## 3. 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 모바일 호환 | iOS Safari, Android Chrome |
| 로딩 속도 | CDN 기반, 서버 API 불필요 (URL 파라미터 방식) |
| 데이터 보관 | localStorage (브라우저 단위) |
| 접근성 | 폰트 14px 이상, 충분한 색상 대비 |
| 인쇄 | @media print로 헤더/푸터 최적화 |

- 다크모드 완전 지원
