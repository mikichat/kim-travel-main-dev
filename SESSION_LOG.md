### ✅ 완료된 작업

#### 1. 사이드바 메뉴 수정
- **파일**: `flight-schedule.html`, `air1/index.html`
- **문제**: 사이드바 메뉴 클릭 시 오류 발생
- **해결**:
  - `toggleSubmenu` 함수 파라미터 수정 (`event` 추가)
  - 일정 관리 메뉴를 서브메뉴 구조로 변경
  - 백업 관리 메뉴 추가

#### 2. 그룹별 일정 / 전체 일정 관리 사이드바 추가

- **문제**: 사이드바가 없어 네비게이션 불편
- **해결**:
  - 사이드바 HTML/CSS/JS 추가
  - 통합 네비게이션 구조

#### 3. 항공편 선택 삭제 기능 수정
- **파일**: `flight-schedule.html`
- **문제**: 선택 삭제 시 localStorage에 반영 안 됨
- **해결**:
  - `FlightSyncManager.deleteFlight()` 사용
  - `sourceId` 기반 실제 데이터 삭제
  - 삭제 후 자동 새로고침

#### 4. 상품관리 팝업 항공편 선택 출력 문제 수정
- **파일**: `js/modules/modals.js`
- **문제**: 항공편 선택 시 정보가 출력되지 않음
- **해결**:
  - 데이터 구조 수정 (`flight.flights` 배열 사용)
  - 출발편/귀국편 정보 올바르게 추출

#### 5. 상품명 → 단체명 변경 및 자동입력
- **파일**: `index.html`, `js/modules/modals.js`
- **개선사항**:
  - "상품명" 레이블 → "단체명"으로 변경
  - 항공편 선택 시 자동입력:
    - ✅ 단체명 (`flight.name`)
    - ✅ 목적지 (첫 번째 항공편 도착지)
    - ✅ 여행기간 (출발일~귀국일 자동계산)
    - ✅ 항공사
    - ✅ 출발편/귀국편 정보
  - Placeholder 추가 ("항공편 선택 시 자동입력됩니다")

#### 6. 문서화
