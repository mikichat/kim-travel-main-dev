**테스트 일자**: 2025-12-29

## 🧪 테스트 체크리스트

### 1단계: 기본 동작 확인

#### ✅ Wizard 모달 열기
- [ ] 상품 관리 섹션에서 **"+ 상품 추가"** 버튼 클릭
- [ ] 모달이 열리면서 **Step 1: 기본 정보 및 항공편** 표시
- [ ] Progress Indicator: `[1●주황] - [2○회색] - [3○회색]`

#### ✅ Step 1 → Step 2 이동
- [ ] 단체명, 목적지, 여행기간, 가격, 상태 **모두 입력**
- [ ] **"다음 단계: 수배 세부사항"** 버튼 클릭
- [ ] Step 2 화면으로 전환됨
- [ ] Progress Indicator: `[1✓초록] - [2●주황] - [3○회색]`
- [ ] **"이전 단계"** 버튼 표시됨

#### ✅ Step 2 → Step 3 이동
- [ ] (입력 선택) 호텔/차량/가이드 정보 입력
- [ ] **"다음 단계: 수배 관리"** 버튼 클릭
- [ ] Step 3 화면으로 전환됨
- [ ] Progress Indicator: `[1✓초록] - [2✓초록] - [3●주황]`
- [ ] **"상품 등록 완료"** 버튼 표시됨

#### ✅ Step 3 → 이전으로 복귀
- [ ] **"이전 단계"** 버튼 클릭 → Step 2로 돌아감
- [ ] 다시 **"이전 단계"** 클릭 → Step 1로 돌아감
- [ ] 입력했던 데이터 유지되는지 확인

---

### 2단계: 필수 필드 검증 (Step 1)

#### ✅ 빈 값 검증
- [ ] 모든 필드를 **비운 상태**에서 "다음 단계" 클릭
- [ ] 빨간색 에러 메시지 박스 표시:
  ```
  다음 항목을 확인해주세요:
  • 단체명을 입력해주세요.
  • 목적지를 입력해주세요.
  • 여행 기간을 입력해주세요.
  • 가격을 입력해주세요.
  ```
- [ ] 각 필드에 **빨간 테두리** 표시됨
- [ ] 5초 후 에러 메시지 자동으로 사라짐

#### ✅ 부분 입력 검증
- [ ] **단체명만** 입력 후 "다음 단계" 클릭
- [ ] 나머지 4개 필드 에러 메시지 표시
- [ ] 단체명 필드는 빨간 테두리 **없음**

#### ✅ 모든 필드 입력 후
- [ ] 단체명, 목적지, 여행기간, 가격, 상태 **모두 입력**
- [ ] "다음 단계" 클릭 시 **에러 없이** Step 2로 이동

---

### 3단계: 항공편 자동입력 (기존 기능 유지)

#### ✅ 항공편 선택 → 자동입력
- [ ] "저장된 항공편 불러오기" 드롭다운 클릭
- [ ] 항공편 1개 선택 (예: "방콕 단체 2024-12-30")
- [ ] **단체명** 자동입력됨
- [ ] **목적지** 자동입력됨 (첫 번째 항공편 도착지)
- [ ] **여행기간** 자동계산됨 (출발일~귀국일)
- [ ] **항공사** 자동입력됨
- [ ] **출발편/귀국편** 자동입력됨

---

### 4단계: UI/UX 디자인 검증 (중장년층 친화)

#### ✅ 색상 시스템
- [ ] Progress Indicator 현재 단계: **주황색 #FF6B35**
- [ ] 완료 단계: **초록색 #10B981**
- [ ] 미완료 단계: **회색 #9CA3AF**
- [ ] "다음 단계" 버튼: **주황색 배경**
- [ ] 체크박스 선택 시: **주황색 테두리 + 배경**

#### ✅ 버튼 크기 (중장년층)
- [ ] 모든 버튼 높이: **48px** (손가락으로 누르기 쉬움)
- [ ] 버튼 텍스트 크기: **16px** 이상
- [ ] 버튼 간격: **12px** (오클릭 방지)

#### ✅ 텍스트 크기
- [ ] 본문 텍스트: **16px**
- [ ] Step 제목: **20px**
- [ ] 레이블: **16px**
- [ ] 필수 표시 (*): **18px 빨간색**

#### ✅ Step Indicator 크기
- [ ] 원형 크기: **56px** (큰 원)
- [ ] 숫자/체크 아이콘: **22px**
- [ ] Step 레이블: **16px**

#### ✅ 체크박스 크기 (Step 3)
- [ ] 체크박스: **24px × 24px** (기존 20px보다 큼)
- [ ] 레이블 텍스트: **16px**

- [ ] 상품 목록에 **새 상품 추가됨**
