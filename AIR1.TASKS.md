## 사용 방법

### 모바일 이미지 저장
1. 항공편 변환기에서 항공편 정보 변환
2. 고객 정보 입력 (이름, 전화번호, 미팅 장소 등)
3. 비용 정보 입력 (선택)
4. **"모바일" 버튼** 클릭
5. PNG 이미지로 자동 다운로드

### 직접 접근
- URL: `http://localhost:5000/air1/reservation.html`
- 데모 데이터가 기본으로 표시됨
