# 📋 견적서 편집기 - 항공편 불러오기 기능 가이드

## ✅ 구현 완료

**견적서 편집기에서 저장된 항공편 정보를 자동으로 불러올 수 있습니다!**

---

## 🎯 주요 기능

### 1. 항공편 드롭다운 선택
- 견적서 편집기 좌측 패널에 **"항공편 선택"** 드롭다운 추가
- localStorage에 저장된 모든 항공편 목록 표시
- 날짜, 이름, 첫 번째 항공편 번호로 구분

### 2. 자동 입력 필드
항공편 선택 시 다음 필드가 **자동으로 입력**됩니다:

| 필드 | 설명 | 예시 |
|------|------|------|
| **항공사명** | 항공사 코드에서 자동 추출 | "아시아나 항공" |
| **출발편** | 출발 공항 → 도착 공항 + 시간 | "인천 (10:30) - 방콕 (14:30)" |
| **귀국편** | 귀국 공항 → 도착 공항 + 시간 | "방콕 (15:30) - 인천 (22:30)" |
| **여행 일자** | 출발일 ~ 귀국일 | "2025년 2월 15일 ~ 2월 20일" |
| **여행지** | 도착 공항명 | "방콕" |

### 3. 지원 항공사

FlightLoader가 자동으로 인식하는 항공사:

```javascript
OZ - 아시아나 항공
KE - 대한항공
TW - 티웨이항공
LJ - 진에어
BX - 에어부산
7C - 제주항공
ZE - 이스타항공
RS - 에어서울
TG - 타이항공
VN - 베트남항공
CZ - 중국남방항공
MU - 중국동방항공
CA - 중국국제항공
NH - ANA
JL - JAL
```

## 🧪 테스트 방법

### 방법 1: 자동 테스트 페이지 (추천)

```
http://localhost:5000/quote-editor-v1/test-flight-loader.html
```

**단계별 테스트:**

1. **사전 요구사항 확인**
   - "사전 요구사항 확인" 버튼 클릭
   - StorageManager, localStorage, FlightLoader 모두 ✓ 확인

2. **샘플 데이터 추가**
   - "샘플 항공편 추가" 버튼 클릭
   - 테스트용 항공편 자동 생성 및 저장

3. **FlightLoader 테스트**
   - "FlightLoader 테스트 실행" 버튼 클릭
   - 항공사명, 출발편, 귀국편, 여행 일자 추출 확인

4. **실제 페이지에서 확인**
   - "견적서 편집기로 이동" 버튼 클릭
   - 드롭다운에서 항공편 선택
   - 자동 입력 확인

## 💻 사용 방법

```

```

### 2. 항공편 선택

좌측 편집 패널 상단에서:

```
┌─────────────────────────────────┐
│ 항공편 선택                      │
│ ┌─────────────────────────────┐ │
│ │ 2025. 2. 15. | 테스트 항공편 │ 인천→방콕 | OZ 747 │
```

선택한 항공편 정보가 즉시 아래 필드에 입력됩니다:
- ✅ 항공사명
- ✅ 출발편
- ✅ 귀국편
- ✅ 여행 일자
- ✅ 여행지

우측 미리보기 패널이 자동으로 업데이트되어 변경사항을 즉시 확인할 수 있습니다.

## 🔧 기술 세부사항

### FlightLoader 클래스

**주요 메서드:**

```javascript
// 1. 항공편 목록 로드
loadFlightOptions()
// → 드롭다운에 저장된 항공편 추가

// 2. 항공편 선택 처리
handleFlightSelection(flightId)
// → 선택된 항공편 데이터 가져오기

// 3. 폼에 자동 입력
populateFormFromFlight(flight)
// → 모든 필드에 데이터 매핑

// 4. 항공사명 추출
extractAirlineName(flight)
// → "OZ" → "아시아나 항공"

// 5. 항공편 경로 포맷팅
formatFlightRoute(flight)
// → "인천 (10:30) - 방콕 (14:30)"

// 6. 여행 일자 계산
calculateTravelDates(firstFlight, lastFlight)
// → "2025년 2월 15일 ~ 2월 20일"
```

### 데이터 흐름

```
항공편 변환기 (air1/index.html)
        ↓
    localStorage
    (StorageManager)
        ↓
견적서 편집기 (quote-editor-v1/index.html)
        ↓
    FlightLoader
        ↓
    자동 입력
```

## 🐛 문제 해결

### 문제 1: 드롭다운이 비어있음

**원인:** 저장된 항공편 데이터가 없음

**해결:**
```
1. http://localhost:5000/air1/index.html 접속
2. 항공편 정보 입력 및 저장
3. 견적서 편집기 새로고침
```

### 문제 2: 선택해도 자동 입력 안 됨

**원인:** StorageManager 또는 FlightLoader 로드 실패

**해결:**
```
1. F12 → Console 탭에서 오류 확인
2. 페이지 새로고침 (Ctrl+F5)
3. test-flight-loader.html에서 사전 요구사항 확인
```

### 문제 3: 항공사명이 코드로 표시됨

**원인:** 항공사 매핑에 없는 코드

**해결:**
```javascript
// flight-loader.js의 airlineMap에 추가
const airlineMap = {
    'OZ': '아시아나 항공',
    'YOUR_CODE': '항공사 이름',  // 추가
    ...
};
```
