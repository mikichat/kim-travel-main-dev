# 🔄 견적서 편집기 - 데이터 동기화 기능 가이드

## ✅ 구현 완료

견적서 편집기(quote-editor-v1/index.html)에 **air1/index.html과의 데이터 동기화 기능**이 추가되었습니다!

---

## 🎯 주요 기능

### 1. **🔄 데이터 동기화 버튼**
- 위치: 우측 미리보기 패널 상단
- "🔄 데이터 동기화" 버튼 클릭 시 모달 창 열림

### 2. **📊 현재 상태 확인**
- 현재 포트 표시
- 저장된 항공편 개수 확인

### 3. **📤 데이터 내보내기**
- 현재 포트의 모든 localStorage 데이터를 JSON으로 내보내기
- 클립보드에 복사 가능

### 4. **📥 데이터 가져오기**
- 다른 포트에서 내보낸 JSON 데이터 붙여넣기
- 자동으로 localStorage에 저장
- 항공편 드롭다운 자동 업데이트

### 5. **⚡ Air1 페이지 바로 열기**
- 버튼 클릭으로 air1/index.html 새 탭에서 열기

---

## 📋 사용 방법

### 시나리오 1: 8000 포트 → 5000 포트 동기화

#### **Step 1: 8000 포트에서 데이터 내보내기**

```
1. http://localhost:8000/quote-editor-v1/index.html 접속
2. 우측 상단 "🔄 데이터 동기화" 버튼 클릭
3. 모달 창에서 "📤 데이터 내보내기" 버튼 클릭
4. "📋 클립보드 복사" 버튼 클릭
```

#### **Step 2: 5000 포트에서 데이터 가져오기**

```
1. http://localhost:5000/quote-editor-v1/index.html 접속
2. 우측 상단 "🔄 데이터 동기화" 버튼 클릭
3. "📥 데이터 가져오기" 섹션의 텍스트 영역에 Ctrl+V로 붙여넣기
4. "📥 데이터 가져오기" 버튼 클릭
5. 확인 대화상자에서 "확인" 클릭
6. ✅ 동기화 완료!
```

---

### 시나리오 2: Air1에서 직접 불러오기

```
1. 견적서 편집기에서 "🔄 데이터 동기화" 버튼 클릭
2. "⚡ Air1에서 불러오기" 섹션에서 "🚀 Air1 페이지 열기" 클릭
3. 새 탭에서 air1/index.html이 열림
4. Air1에서 항공편 데이터 저장
5. 견적서 편집기로 돌아와서 페이지 새로고침 (F5)
6. 항공편 드롭다운에 새 데이터 표시
```

## ✨ 개선 사항

### 추가된 UI 요소

1. **동기화 버튼**
   - 위치: 미리보기 패널 우측 상단
   - 색상: 파란색 (브랜드 색상)
   - 아이콘: 🔄

2. **모달 창**
   - 크기: 최대 700px 너비
   - 반응형: 모바일 대응 (90% 너비)
   - 스크롤: 내용이 많을 경우 자동 스크롤

3. **상태 표시**
   - 성공: 녹색 배경
   - 정보: 파란색 배경
   - 경고: 노란색 배경

### 추가된 JavaScript 함수

```javascript
// 모달 제어
openSyncModal()       // 모달 열기
closeSyncModal()      // 모달 닫기
updateSyncStatus()    // 현재 상태 업데이트

// 데이터 동기화
exportSyncData()      // 데이터 내보내기
copySyncData()        // 클립보드 복사
importSyncData()      // 데이터 가져오기
clearSyncImport()     // 입력란 지우기

// 유틸리티
openAir1()            // Air1 페이지 열기
```

## 🔍 문제 해결

### 문제 1: 모달이 열리지 않음

**원인:** JavaScript 오류

**해결:**
```
F12 → Console 탭에서 오류 확인
페이지 새로고침 (Ctrl+F5)
```

### 문제 2: 클립보드 복사 안 됨

**원인:** 브라우저 권한 문제

**해결:**
```
HTTPS가 아닌 경우 수동 복사 (Ctrl+A → Ctrl+C)
```

### 문제 3: 가져오기 후 드롭다운 업데이트 안 됨

**원인:** FlightLoader 미로드

**해결:**
```
페이지 새로고침 (F5)
또는 모달 닫고 다시 열기
```

### 문제 4: JSON 파싱 오류

**원인:** 잘못된 JSON 형식

**해결:**
```
내보낸 전체 내용을 복사했는지 확인
JSON 검증 (jsonlint.com)
```

### 데이터 흐름

```
Air1 (항공편 변환기)
    ↓ 저장

    ↓ 동기화 모달
견적서 편집기
    ↓ 내보내기

    ↓ 붙여넣기

    ↓ 가져오기

    ↓

```
