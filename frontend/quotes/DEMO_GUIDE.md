## 🎬 데모 시나리오

### 시나리오 1: 새로운 여행 패키지 작성
**목표**: 일본 여행 패키지 견적서 + 일정표 작성

#### Step 1: 편집기 열기
```
1. index-full.html 파일을 브라우저에서 열기
2. 좌측 편집 패널과 우측 미리보기 확인
3. 기본 데이터가 로드되어 있는지 확인
```

#### Step 2: 견적서 작성
```
[견적서 탭 클릭]

단체 정보:
- 단체명: "일본 – 도쿄 오사카 5일"
- 여행 일자: "2025년 12월 25일 ~ 12월 29일"
- 여행지: "일본 – 도쿄 오사카"

요금 정보:
- 항공사명: "대한 항공"
- 출발편: "인천 (09:00) - 나리타 (11:30)"
- 귀국편: "간사이 (13:00) - 인천 (15:30)"
- 옵션: "전일옵션"
- 1인 요금: "2,450,000원"

[미리보기 업데이트 클릭]
→ 우측 미리보기에서 견적서 확인
```

#### Step 3: 일정표 작성 (1일차)
```
[일정표 탭 클릭]
[1일차 버튼 선택]

일차 정보:
- 일차: 1
- 날짜: "12/25"
- 요일: "수"
- 일정 제목: "인천 → 도쿄"

일정 내용:
- 주요 일정:
  09:00 인천국제공항 출발
  11:30 나리타 공항 도착
  오후 도쿄 시내 관광
  저녁 아사쿠사 탐방

- 식사: "조: 기내식 / 중: 현지식 / 석: 한식"
- 숙박: "도쿄 ****호텔"

[미리보기 업데이트 클릭]
→ 우측에서 일정표 페이지 추가 확인
```

#### Step 4: 일정표 작성 (2일차)
```
[2일차 버튼 선택]

일차 정보:
- 일차: 2
- 날짜: "12/26"
- 요일: "목"
- 일정 제목: "도쿄 → 후지산 → 하코네"

일정 내용:
- 주요 일정:
  조식 후 호텔 출발
  오전 후지산 5합목 관광
  오후 하코네 온천마을
  저녁 온천 체험

- 식사: "조: 호텔식 / 중: 현지식 / 석: 료칸식"
- 숙박: "하코네 온천 료칸"

[미리보기 업데이트 클릭]
```

#### Step 5: 저장 및 출력
```
[전체 저장 클릭]
→ "전체 문서가 저장되었습니다" 메시지 확인

[전체 인쇄/PDF 클릭]
→ 브라우저 인쇄 대화상자 열림
→ "PDF로 저장" 선택
→ 파일명: "일본_여행_견적서_2025-12-25.pdf"
```

## 🎯 데모 시나리오 2: 데이터 수정 및 재사용

### Step 1: 저장된 문서 불러오기
```
1. index-full.html 열기
2. [불러오기] 클릭
3. 이전에 저장한 데이터 자동 로드 확인
4. 견적서와 일정표 모두 복원 확인
```

### Step 2: 견적서 수정
```
[견적서 탭]
- 단체명 변경: "일본 – 도쿄 오사카 5일" → "일본 – 교토 나라 6일"
- 요금 변경: "2,450,000원" → "2,650,000원"
[미리보기 업데이트]
```

### Step 3: 일정표 추가
```
[일정표 탭]
[3일차 버튼 선택 또는 + 추가]

3일차 정보:
- 날짜: "12/27"
- 요일: "금"
- 제목: "하코네 → 오사카"
- 일정: 조식 후 오사카 이동...

[미리보기 업데이트]
→ 3페이지 확인
```

### Step 4: 재저장
```
[전체 저장]
→ 업데이트된 문서 LocalStorage에 저장
```

## 🐛 예상 이슈 및 해결

### 이슈 1: 미리보기가 업데이트되지 않음
**증상**: "미리보기 업데이트" 클릭 시 변경사항이 반영되지 않음

**해결책**:
1. 브라우저 콘솔 확인 (F12)
2. 에러 메시지 확인
3. 페이지 새로고침 (Ctrl+R)
4. 데이터 다시 입력

### 이슈 2: 일정표 배경 이미지 표시 안됨
**증상**: 일정표 페이지에 배경 이미지가 없음

**원인**: `itinerary-data.js`의 `SIMPLE_ITINERARY_DATA`에서 `background_image` 경로 미설정

**해결책**:
```javascript
// 현재 (이미지 없음)
background_image: ''

// 수정 (이미지 경로 지정)
background_image: '../itinerary/chain/can/25년11월중순 광저우 망산-1_hd5.png'
```

**참고**: Phase 2에서는 배경 이미지 없이 텍스트만 오버레이하는 방식으로 구현됨

### 이슈 3: PDF 출력 시 색상 누락
**증상**: PDF로 저장 시 배경색이나 강조색이 나타나지 않음

**해결책**:
1. Chrome: 인쇄 설정 → "배경 그래픽" 체크
2. Firefox: 페이지 설정 → "배경색 및 이미지 인쇄" 체크
3. Edge: 기타 설정 → "배경 그래픽" 체크

### 이슈 4: LocalStorage 데이터 손실
**증상**: 저장 후 브라우저 종료 시 데이터 사라짐

**원인**:
- 시크릿 모드 사용
- 브라우저 설정에서 쿠키 자동 삭제

**해결책**:
1. 일반 모드에서 사용
2. 브라우저 설정에서 LocalStorage 허용
3. 중요 데이터는 JSON 파일로 백업
