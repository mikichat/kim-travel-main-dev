## 🚀 사용 방법 (Quick Start)

### 1. 기본 사용법
```
1. index-full.html 파일을 브라우저에서 열기
2. 좌측 "견적서" 탭: 단체 정보 및 요금 입력
3. 좌측 "일정표" 탭: 일차별 일정 입력
4. "미리보기 업데이트" 클릭
5. 우측에서 전체 문서 확인
6. "전체 저장" → "전체 인쇄/PDF" 클릭
```

### 2. 권장 워크플로우
```
견적서 작성 → 일정표 작성 → 미리보기 → 저장 → PDF 출력
```
