```javascript
{
  document_id: 'DOC-xxx',
  quote: {
    // Phase 1 견적서 데이터
  },
  itinerary: {
    pages: [
      {
        page_number: 2,
        page_type: 'itinerary_day',
        day_info: { ... },
        editable_text: { ... },
        background_image: '...'
      }
    ]
  }

```

### 2. 페이지 구조
```html
<!-- 일정표 페이지 -->
<div class="itinerary-page-container">
  <!-- 배경 이미지 레이어 (z-index: 1) -->
  <div class="itinerary-background"></div>

  <!-- 콘텐츠 레이어 (z-index: 2) -->
  <div class="itinerary-content">
    <div class="itinerary-header">...</div>
    <div class="schedule-section">...</div>
    <div class="meals-section">...</div>
    <div class="hotel-section">...</div>
  </div>
</div>
```

```

  ↓ renderQuotePage() + renderItineraryPages()
화면 표시
```

## 🔄 시스템 사용 흐름

### 견적서 + 일정표 작성
1. `index-full.html` 열기
2. **견적서 탭**: 단체 정보, 요금 정보 입력
3. **일정표 탭**: 일차 선택 → 일정 내용 입력
4. **미리보기 업데이트** 클릭
5. 우측에서 전체 문서 확인
6. **전체 저장** 클릭
7. **전체 인쇄/PDF** 클릭

### 저장된 문서 불러오기
1. `index-full.html` 열기
2. **불러오기** 클릭
3. 자동으로 모든 데이터 복원
4. 필요 시 수정
5. **미리보기 업데이트** → **전체 저장**
