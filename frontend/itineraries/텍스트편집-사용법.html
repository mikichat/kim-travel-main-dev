📝 글자 단위별 색상 변경 & 강조 기능

미리보기 페이지에서 텍스트를 자유롭게 편집하세요!

✨ 사용 가능한 기능

굵게 (Bold)
선택한 텍스트를 굵게 표시
글자 색상 변경
원하는 색상으로 글자 색 변경
배경 색상 (형광펜)
중요한 텍스트를 형광펜으로 강조
글자 크기 변경
글자 크기를 8px ~ 72px로 조절

🎯 사용 방법 (3단계)

1

미리보기 & 공유하기 버튼 클릭

travel-advanced.html에서 입력을 완료한 후 "미리보기 & 공유하기" 버튼을 클릭하여 preview.html 페이지로 이동합니다.

2

텍스트를 마우스로 드래그하여 선택

편집하고 싶은 글자를 마우스로 드래그하여 선택합니다.

팁: 한 글자만 선택해도 되고, 단어, 문장 전체를 선택해도 됩니다!

3

자동으로 나타나는 툴바에서 원하는 기능 선택

텍스트를 선택하면 편집 툴바가 자동으로 나타납니다.

👆 이런 모양의 툴바가 선택한 텍스트 근처에 나타납니다

🔧 각 버튼 기능 설명

굵게

선택한 텍스트를 굵게 표시합니다.

기울임

선택한 텍스트를 기울임으로 표시합니다.

밑줄

선택한 텍스트에 밑줄을 추가합니다.

글자 색상

첫 번째 색상 선택기를 클릭하여 글자 색상을 변경합니다.

배경 색상 (형광펜)

두 번째 색상 선택기로 배경색을 추가합니다.

글자 크기

숫자 입력 후 버튼 클릭하여 크기 변경

서식 지우기

선택한 텍스트의 모든 서식을 제거합니다.

편집 가능 영역

마우스를 올리면 점선 테두리가 나타나는 모든 영역을 편집할 수 있습니다.

💡 실전 예시

예시 1: 중요한 문구 강조하기

여행 일정: 2025년 1월 15일 ~ 1월 20일

➡️ "2025년 1월 15일 ~ 1월 20일"을 선택 → 글자 색상을 빨간색으로 → 굵게 → 크기를 20으로 설정

예시 2: 형광펜으로 중요 정보 표시

출발 시간: 오전 8시 30분 인천공항 제1터미널

➡️ "오전 8시 30분 인천공항 제1터미널"을 선택 → 배경 색상을 노란색으로 → 굵게

예시 3: 여러 스타일 조합하기

특별 혜택! 조기 등록 시 10% 할인

➡️ 각 부분을 선택하여 다양한 스타일 적용

주의사항

  • 미리보기 페이지(preview.html)에서만 텍스트 편집이 가능합니다
  • 편집 내용을 저장하려면 브라우저의 인쇄 기능(Ctrl+P) 또는 "고해상도 이미지 저장" 버튼을 사용하세요
  • 페이지를 새로고침하면 편집 내용이 사라지므로 주의하세요
  • 텍스트를 선택해야만 툴바가 나타납니다

유용한 단축키

Ctrl + B 굵게
Ctrl + I 기울임
Ctrl + U 밑줄
Ctrl + P 인쇄 (저장)
