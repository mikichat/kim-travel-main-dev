localStorage 포트 간 동기화 도구

문제 진단

현재 접속 중인 포트:

문제: localhost:8000과 localhost:5000은 서로 다른 origin으로 localStorage가 격리되어 있습니다.

결과: 한 포트에서 저장한 데이터는 다른 포트에서 접근할 수 없습니다.

해결 방법

아래 도구를 사용하여 데이터를 내보내고 가져오기 할 수 있습니다:

  1. 8000 포트에서 이 페이지를 열고 "데이터 내보내기" 클릭
  2. 나타난 JSON 텍스트를 복사
  3. 5000 포트에서 이 페이지를 열고 복사한 내용을 붙여넣기
  4. "데이터 가져오기" 클릭

1. 현재 포트 상태 확인

2. 데이터 내보내기 (Export)

현재 포트의 모든 localStorage 데이터를 JSON으로 내보냅니다.

3. 데이터 가져오기 (Import)

다른 포트에서 내보낸 JSON 데이터를 여기에 붙여넣고 가져오기를 클릭하세요.
