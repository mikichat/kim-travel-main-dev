# 항공편 변환기 개선 계획

## 목표
항공편 정보 자동 변환 및 관리 기능을 개선하여 여행사 업무 효율성을 높입니다.

## 현재 문제점

### 1. 이름 자동 변환 문제
- **현상**: 이름 뒤 호칭(MR/MS/MSTR/MISS) 제거 안됨
- **요구사항**: 
  - 호칭 자동 제거
  - 첫 번째 이름을 대표자로 자동 설정
  - 총 인원수 자동 계산
  - **두 가지 예약 형식 모두 지원**:
    - 형식 1: `1.KIM/YONGKUMR` (기존)
    - 형식 2: `1.1SO/JAEWANMR` (새로운 형식)

### 2. 항공편 스케줄 관리 문제
- **선택 삭제 오류**: 체크박스 선택 후 삭제 기능 미작동
- **왕복 저장 오류**: 왕복 항공권인데 편도만 저장됨
- **수정 저장 오류**: 수정된 내용이 저장되지 않음

### 3. 팀 통합 기능 부재
- **현상**: 여러 예약번호(PNR)를 가진 팀을 하나로 관리 불가
- **요구사항**: 여러 예약을 하나의 팀으로 통합하는 기능

## 제안 해결방안

### 1️⃣ 이름 자동 변환 개선

#### 수정 파일: `air1/js/main.js`

**변경 내용:**
- `handleConvert()` 함수에 이름 파싱 로직 추가
- 두 가지 형식 모두 처리하는 정규식 패턴 사용
- 호칭(MR/MS/MSTR/MISS) 제거 함수 추가
- 첫 번째 이름을 `nameInput`에 자동 입력
- 총 인원수를 `totalPeopleInput`에 자동 입력

```javascript
// 추가될 로직 예시
function cleanPassengerName(name) {
    return name.replace(/(MR|MS|MSTR|MISS)$/i, '').trim();
}

function parsePassengerNames(inputText) {
    // 형식 1: 1.KIM/YONGKUMR
    // 형식 2: 1.1SO/JAEWANMR (숫자.숫자+이름 형식)
    const nameRegex = /(\d+)\.(\d*[A-Z][A-Z/]+)/g;
    const names = [];
    let match;
    
    while ((match = nameRegex.exec(inputText)) !== null) {
        let fullName = match[2];
        
        // 형식 2의 경우 앞의 숫자 제거 (예: 1SO -> SO)
        fullName = fullName.replace(/^\d+/, '');
        
        names.push({
            index: parseInt(match[1]),
            name: cleanPassengerName(fullName)
        });
    }
    
    return names.sort((a, b) => a.index - b.index);
}
```

### 3️⃣ 팀 통합 기능 추가

#### 수정 파일: `air1/js/saved-flights.js`

**변경 내용:**
- "선택 통합" 버튼 추가
- 선택된 여러 항공편을 하나로 병합하는 함수 구현
- 병합 시 다음 정보 통합:
  - PNR: 쉼표로 구분하여 연결
  - 항공편: 모든 구간 합침
  - 탑승객 이름: 쉼표로 구분하여 연결
  - 총 인원: 합산

```javascript
function mergeSelectedFlights() {
    const selectedIds = getSelectedFlightIds();
    const flights = selectedIds.map(id => StorageManager.getFlightById(id));
    
    const mergedFlight = {
        name: flights[0].name + ' (통합)',
        pnr: flights.map(f => f.pnr).filter(Boolean).join(', '),
        flights: flights.flatMap(f => f.flights),
        customerInfo: {
            name: flights.map(f => f.customerInfo?.name).filter(Boolean).join(', '),
            totalPeople: flights.reduce((sum, f) => {
                return sum + (parseInt(f.customerInfo?.totalPeople) || 0);
            }, 0).toString()
        }
    };
    
    StorageManager.saveFlightData(mergedFlight);
}
```
