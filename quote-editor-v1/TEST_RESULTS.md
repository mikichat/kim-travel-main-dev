## 🐛 발견된 이슈

### 이슈 #1: 일차 추가 버튼 미구현
**심각도**: 낮음
**설명**: "+ 추가" 버튼이 UI에 있지만 클릭 시 새 일차가 추가되지 않음
**재현 방법**:
1. 일정표 탭 클릭
2. "+ 추가" 버튼 클릭
3. 아무 반응 없음

**권장 조치**:
- 동적 일차 추가 기능 구현
- 또는 "+ 추가" 버튼 제거

---

### 이슈 #2: 여러 일차 데이터 저장/복원 미완성
**심각도**: 중간
**설명**: 여러 일차를 작성하고 저장 시 현재 편집 중인 일차만 저장됨
**재현 방법**:
1. 1일차 데이터 입력
2. 2일차 선택 후 데이터 입력
3. 전체 저장
4. 페이지 새로고침 후 불러오기
5. 2일차 데이터만 로드됨 (1일차 손실)

**권장 조치**:
- 모든 일차 데이터를 배열로 저장하는 로직 추가
- 일차 전환 시 자동 저장 기능

---

### 이슈 #3: 일정표 배경 이미지 경로 미설정
**심각도**: 낮음
**설명**: 일정표 페이지에 배경 이미지가 표시되지 않음
**상태**: **의도된 동작** (Phase 2에서 텍스트 오버레이만 구현)

**참고**:
- `itinerary-data.js`의 `background_image` 필드가 빈 문자열
- 실제 배경 이미지를 사용하려면 경로 지정 필요

**권장 조치** (Phase 3):
- 배경 이미지 경로 설정 UI 추가
- 이미지 파일 선택 기능 구현
