# 🔄 데이터 동기화 - 상세 설명

## 📊 동기화하는 데이터

### 개요

**동기화 기능은 `localStorage`에 저장된 모든 데이터를 다른 포트/환경으로 복사합니다.**

---

## 🗂️ 동기화되는 데이터 종류

### 1. **항공편 데이터** (`flight_saves_v2`)

**출처:** Air1 항공편 변환기
**용도:** 견적서 편집기, 일정표 작성

**데이터 내용:**
```javascript
{
  "flight_saves_v2": [
    {
      "id": "FLIGHT-1704038400000",
      "name": "방콕 여행",
      "pnr": "ABC123",
      "saveDate": "2025-01-15T10:00:00Z",
      "flights": [
        {
          "flightNumber": "OZ 747",
          "airline": "OZ",
          "date": "2025.01.20(월)",
          "departure": {
            "airport": "인천",
            "code": "ICN",
            "time": "10:30"
          },
          "arrival": {
            "airport": "방콕",
            "code": "BKK",
            "time": "14:30"
          }
        }
      ],
      "customerInfo": {
        "name": "홍길동",
        "phone": "010-1234-5678",
        "meetingPlace": "인천공항 제1터미널 3층",
        "meetingTime": "2025.01.20 06:30",
        "totalPeople": "45"
      }
    }
  ]
}
```

**포함 정보:**
- ✈️ 항공편 번호 (OZ 747, KE 123 등)
- 🛫 출발/도착 공항 및 시간
- 📅 여행 날짜
- 👤 고객 정보 (이름, 연락처)
- 📍 미팅 장소 및 시간
- 👥 탑승 인원

---

**용도:** 버스 예약 정보 관리

**데이터 내용:**

```javascript
        "datetime": "2025.01.20 14:00"
      },
      "arrival": {
        "location": "서울 강남",
        "datetime": "2025.01.20 15:30"
      },
      "busInfo": {
        "type": "45인승",
        "count": 1
      },
      "passengers": 40,
      "showDriverContact": true,
      "driverContact": "010-9999-8888",
      "showCustomerInfo": true,
      "customerInfo": {
        "name": "김철수",
        "phone": "010-1111-2222"
      },
      "remarks": "공항 픽업"

```

**포함 정보:**
- 🚌 버스 타입 (25/28/45인승)
- 📍 출발지/도착지
- ⏰ 출발/도착 시간
- 👥 탑승 인원

- 👤 대표 고객 정보

### 3. **안내문 데이터** (`saved_notices`)

**출처:** Air1 항공편 변환기 - 안내문 작성 탭
**용도:** 여행 안내문 관리

**데이터 내용:**
```javascript
{
  "saved_notices": [
    {
      "id": "NOTICE-1704038400000",
      "saveDate": "2025-01-15T10:00:00Z",
      "template": "pre-departure",
      "variables": {
        "단체명": "방콕 여행",
        "출발일": "2025.01.20(월)",
        "항공편": "OZ 747",
        "미팅장소": "인천공항 제1터미널",
        "담당자": "홍길동"
      },
      "content": "안녕하세요, 방콕 여행을 담당하고 있는 홍길동입니다..."
    }
  ]
}
```

**포함 정보:**
- 📝 템플릿 종류 (출발 전 안내, 공항 미팅, 준비물)
- 🏷️ 변수 값 (단체명, 날짜, 항공편 등)
- 📄 작성된 안내문 내용

### 4. **견적서 데이터** (`quote_data`)

**출처:** 견적서 편집기

**데이터 내용:**

```javascript
      "destination": "방콕",
      "travel_dates": "2025년 1월 20일 ~ 1월 25일",
      "duration": "4박 5일"
    },
    "pricing": {
      "airline_name": "아시아나 항공",
      "departure_flight": "인천 (10:30) - 방콕 (14:30)",
      "return_flight": "방콕 (15:30) - 인천 (22:30)",
      "price_amount": "1,540,000원",
      "price_note": "성인 1인 기준"
    },
    "contact_info": "홍길동 팀장 (010-1234-5678)"
  }

```

**포함 정보:**
- 📋 단체 정보 (단체명, 목적지)
- ✈️ 항공편 정보
- 💰 가격 정보
- 📞 담당자 연락처

## 🔄 동기화 방향

### A. Air1 → 견적서 편집기

```
┌─────────────────────┐
│  Air1               │
│  항공편 변환기       │
├─────────────────────┤
│ • 항공편 입력/변환   │
│ • 버스 예약 작성     │
│ • 안내문 작성        │
│ • 저장 버튼 클릭     │
└──────────┬──────────┘
           │ 저장
           ▼
    ┌──────────────┐
    │ localStorage │
    │ (포트: 8000) │
    └──────┬───────┘
           │ 동기화 (내보내기 → 가져오기)
           ▼
    ┌──────────────┐
    │ localStorage │
    │ (포트: 5000) │
    └──────┬───────┘
           │ 불러오기
           ▼
┌─────────────────────┐
│  견적서 편집기       │
├─────────────────────┤
│ • 항공편 선택 드롭다운│
│ • 자동 입력          │
│ • 견적서 작성        │
└─────────────────────┘
```

---

### B. 견적서 편집기 → Air1

```
┌─────────────────────┐
│  견적서 편집기       │
│  (포트: 5000)       │
├─────────────────────┤
│ • 견적서 저장        │
│ • 동기화 버튼 클릭   │
│ • 데이터 내보내기    │
└──────────┬──────────┘
           │ 내보내기 → 복사
           ▼
    [클립보드 복사]
           │
           ▼ 붙여넣기
┌─────────────────────┐
│  Air1               │
│  (포트: 8000)       │
├─────────────────────┤
│ • 데이터 가져오기    │
│ • localStorage 저장 │
│ • 데이터 사용        │
└─────────────────────┘
```

## 💡 사용 예시

### 예시 1: 새 항공편 추가 후 견적서 작성

```
1️⃣ Air1 (8000 포트)
   - 항공편 정보 입력
   - 변환 → 저장
   - localStorage에 flight_saves_v2 저장

2️⃣ 동기화
   - 견적서 편집기 (5000 포트) 열기
   - 🔄 데이터 동기화 버튼 클릭
   - 8000 포트에서 데이터 내보내기 → 복사
   - 5000 포트에서 붙여넣기 → 가져오기

3️⃣ 견적서 편집기 (5000 포트)
   - 항공편 선택 드롭다운에 새 항공편 표시
   - 선택하면 자동 입력
   - 견적서 완성
```

---

### 예시 2: 백업 및 복구

```
1️⃣ 백업
   - 🔄 데이터 동기화 클릭
   - 📤 데이터 내보내기
   - JSON 텍스트 복사 → 메모장에 저장
   - 파일명: backup_2025-01-15.json

2️⃣ 복구
   - localStorage 초기화 (실수 또는 의도적)
   - 🔄 데이터 동기화 클릭
   - 백업 파일 열기 → JSON 복사
   - 📥 데이터 가져오기 → 붙여넣기
   - ✅ 데이터 복구 완료
```

---

### 예시 3: 포트 간 작업 전환

```
개발 중: 8000 포트
실제 업무: 5000 포트

1️⃣ 개발 완료
   - 8000 포트에서 데이터 내보내기

2️⃣ 실 업무 환경으로 이동
   - 5000 포트에서 데이터 가져오기

3️⃣ 결과
   - 동일한 항공편, 버스예약, 안내문 데이터 사용 가능
```

## 🎯 핵심 정리

### 동기화하는 데이터

| 데이터 | 출처 | 용도 |
|--------|------|------|
| **항공편** | Air1 | 견적서, 일정표 자동 입력 |
| **버스예약** | Air1 | 버스 예약 관리 |
| **안내문** | Air1 | 여행 안내문 작성 |
| **견적서** | 견적서 편집기 | 견적서 저장 |

### 동기화 방향

```
Air1 (8000) ←→ 견적서 편집기 (5000)
      ↕
localStorage 데이터
```

### 동기화 범위

**모든 localStorage 데이터 = 완전 동기화**

- ✅ 항공편 모두
- ✅ 버스예약 모두
- ✅ 안내문 모두
- ✅ 견적서 데이터
- ✅ 기타 설정

## ❓ 자주 묻는 질문

### Q1: 일부 데이터만 동기화할 수 있나요?

**A:** 현재 버전은 **전체 동기화**만 지원합니다.
내보내기 시 모든 localStorage 데이터가 포함됩니다.

---

### Q2: 동기화하면 기존 데이터는 어떻게 되나요?

**A:** **덮어쓰기** 됩니다.
- 가져오기 전에 확인 대화상자 표시
- 기존 데이터는 사라짐
- 백업 권장

---

### Q3: 자동 동기화는 안 되나요?

**A:** 브라우저 보안 정책상 **자동 동기화 불가능**합니다.
- 포트가 다르면 별도 origin
- localStorage 격리됨
- 수동 복사 필요

---

### Q4: 어떤 상황에서 동기화를 사용하나요?

**A:** 다음 상황에서 사용:
1. **포트 변경**: 8000 → 5000
2. **백업**: 정기적 데이터 저장
3. **복구**: 데이터 손실 후 복원
4. **협업**: 팀원 간 데이터 공유
5. **환경 전환**: 개발 → 운영
