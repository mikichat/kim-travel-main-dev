# Available Skills

The following skills are available to assist you. Run them using the commands below:

1.  **/socrates**
    *   **Description**: Socrates-style 1:1 consulting to refinement ideas into structured planning documents (PRD, TRD, etc.).
    *   **Use when**: You have a rough idea and need a detailed plan.

2.  **/project-bootstrap**
    *   **Description**: Scaffolds a new project with your chosen tech stack (Frontend, Backend, Database).
    *   **Use when**: You want to start a new project or set up agent teams.

3.  **/orchestrate**
    *   **Description**: Orchestrates complex tasks by breaking them down and executing them.
    *   **Use when**: You have a large, complex task that needs to be managed step-by-step.

4.  **/deep-research**
    *   **Description**: Deep research skill for thorough investigation and analysis.
    *   **Use when**: You need comprehensive research on a topic.

5.  **/tasks-generator**
    *   **Description**: Generates task lists and breakdowns.
    *   **Use when**: You need to create detailed task lists from requirements.
