## 📋 일일 체크리스트 (요약)

작업 시작할 때 매번 확인:

```
□ SESSION_LOG 읽기 (어디까지 했나?)
□ ROADMAP 확인 (다음 할 일은?)
□ 작업 범위 명확히 (Non-goals 체크)
□ 파일 읽고 수정 (Read → Edit)
□ 에러 처리 추가
□ 테스트 (정상 케이스 + 에러 케이스)
□ SESSION_LOG 업데이트
□ 버전 정보 업데이트
□ 다음 작업 계획
```
