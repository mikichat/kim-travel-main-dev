# 상품 관리 시스템 사용 가이드

## ✅ 완료된 작업

### 1. 데이터베이스 구조 업데이트
- **파일**: `backend/travel_agency.db`
- **테이블**: `products`
- **추가된 컬럼**: 27개

#### 📋 추가된 필드

**✈️ 항공편 정보**
- `flight_id` - 저장된 항공편 ID
- `airline` - 항공사명
- `outbound_flight` - 출발편 정보
- `return_flight` - 귀국편 정보
- `flight_note` - 항공 비고

**🏨 호텔 정보**
- `hotel_name` - 호텔명
- `hotel_checkin` - 체크인 날짜
- `hotel_checkout` - 체크아웃 날짜
- `hotel_room_type` - 객실 타입
- `hotel_rooms` - 객실 수
- `hotel_note` - 호텔 비고

**🚌 차량 정보**
- `vehicle_type` - 차량 종류
- `vehicle_count` - 차량 대수
- `vehicle_company` - 차량 업체
- `vehicle_note` - 차량 비고

**👤 가이드 정보**
- `guide_name` - 가이드명
- `guide_phone` - 연락처
- `guide_language` - 가이드 언어
- `guide_note` - 가이드 비고

**📋 수배업무**
- `procurement_flight` - 항공권 수배 (체크박스)
- `procurement_hotel` - 호텔 예약 (체크박스)
- `procurement_vehicle` - 차량 예약 (체크박스)
- `procurement_guide` - 가이드 섭외 (체크박스)
- `procurement_visa` - 비자 수속 (체크박스)
- `procurement_insurance` - 보험 가입 (체크박스)
- `procurement_status` - 수배 진행 상태 (대기/진행중/완료)
- `procurement_note` - 수배 메모

## 🎯 사용 방법

### 1️⃣ 상품 관리 페이지 접속

```
http://localhost:8000/index.html#products
```

또는 왼쪽 사이드바에서 **"상품 관리"** 클릭

### 2️⃣ 상품 진행 (수정)

1. 상품 목록에서 원하는 상품(예: 베트남 - 하노이 하롱베이) 클릭
2. **"상품 진행"** 모달 팝업이 열림
3. 다음 정보 입력:

#### 📋 기본 정보
- 상품명, 목적지, 여행 기간, 가격, 상태, 상품 설명

#### ✈️ 항공편 정보
- **저장된 항공편 불러오기**: air1에서 저장한 항공편 선택
- 선택 시 자동으로 항공사, 출발편, 귀국편 정보 입력됨
- 수동 입력도 가능

#### 🏨 호텔 정보
- 호텔명, 체크인/체크아웃 날짜
- 객실 타입, 객실 수
- 호텔 비고

#### 🚌 차량 정보
- 차량 종류(예: 대형버스 45인승), 차량 대수
- 차량 업체
- 차량 비고

#### 👤 가이드 정보
- 가이드명, 연락처
- 가이드 언어
- 가이드 비고

#### 📋 수배업무 관리
- **체크리스트**: 항공권, 호텔, 차량, 가이드, 비자, 보험
- **수배 진행 상태**: 수배 대기 / 수배 진행중 / 수배 완료
- **수배 메모**: 수배업무 관련 메모

### 3️⃣ 저장 및 단체명단 확인

모달 하단의 3개 버튼:

- **취소**: 변경사항을 저장하지 않고 모달 닫기
- **📋 단체명단 보기**: 단체명단 페이지로 이동 (모달 닫힘)
- **💾 저장**: 모든 정보를 데이터베이스에 저장

## 📊 데이터 저장 위치

### SQLite 데이터베이스
- **파일 경로**: `backend/travel_agency.db`
- **테이블**: `products`
- **데이터 확인 방법**:
  ```bash
  cd backend
  sqlite3 travel_agency.db

  # 테이블 구조 확인
  .schema products

  # 데이터 조회
  SELECT * FROM products;

  # 특정 상품 조회
  SELECT name, airline, hotel_name, procurement_status
  FROM products
  WHERE name LIKE '%베트남%';
  ```

## 🔍 단체명단 확인 방법

### 방법 1: 상품 진행 모달에서
1. 상품 진행 모달 열기
2. 하단의 **"📋 단체명단 보기"** 버튼 클릭
3. 자동으로 단체명단 페이지로 이동

### 방법 2: 사이드바에서
1. 왼쪽 사이드바에서 **"단체명단"** 클릭
2. 단체명단 페이지 열림
3. 검색창에 상품명 입력하여 검색

### 방법 3: 직접 URL
```
http://localhost:8000/index.html#group-roster
```

## 🔄 데이터 흐름

```
1. 항공권 관리 (air1)
   ↓ 항공편 저장 (localStorage)

2. 상품 관리
   ↓ 저장된 항공편 선택
   ↓ 호텔, 차량, 가이드, 수배업무 입력
   ↓ 💾 저장 버튼 클릭
   ↓ 데이터베이스에 저장 (SQLite)

3. 단체명단
   ↓ 상품과 연결된 단체 확인
   ↓ 고객 명단 관리
```

## 📝 참고사항

### 체크박스 값 저장
- JavaScript에서 `checked` 속성은 `true/false`로 전달
- SQLite에는 `1/0` (INTEGER)로 저장
- 불러올 때 자동 변환됨

### 날짜 형식
- 입력: `<input type="date">` → `YYYY-MM-DD`
- 저장: TEXT 형식
- 예: `2025-12-28`

### 항공편 연결
- `flight_id`에 air1의 항공편 ID 저장
- 필요 시 localStorage에서 상세 정보 조회 가능
